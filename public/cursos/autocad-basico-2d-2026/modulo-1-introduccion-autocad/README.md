# Módulo 1: Introducción a AutoCAD 2026

## Descripción del Módulo

Este módulo fundamental introduce al estudiante en el entorno de trabajo de AutoCAD 2026, proporcionando las bases necesarias para comenzar a crear dibujos técnicos de manera eficiente. Se explora la interfaz renovada de AutoCAD 2026, los sistemas de coordenadas, métodos de navegación y las herramientas esenciales que servirán como fundamento para todos los módulos posteriores.

El enfoque está en desarrollar fluidez con la interfaz, comprender el sistema de coordenadas cartesianas que AutoCAD utiliza, y dominar los métodos de entrada de datos que maximizan la precisión y productividad. También se aprende a gestionar archivos DWG y crear plantillas personalizadas que acelerarán proyectos futuros.

## Objetivos de Aprendizaje

Al completar este módulo, el estudiante será capaz de:

1. **Navegar eficientemente** en la interfaz de AutoCAD 2026, utilizando ribbons, paletas y espacios de trabajo
2. **Configurar** el entorno de trabajo según estándares profesionales y preferencias personales
3. **Aplicar** el sistema de coordenadas absolutas, relativas y polares para dibujo preciso
4. **Utilizar** comandos de zoom y pan para navegar en dibujos de cualquier escala
5. **Seleccionar objetos** mediante múltiples métodos (individual, ventana, crossing, fence)
6. **Implementar** entrada dinámica para entrada rápida de datos y coordenadas
7. **Gestionar archivos** DWG, crear plantillas y configurar unidades de trabajo

- Conocimientos básicos de computación (Windows, manejo de archivos)

## Contenido del Módulo

### Lección 1: Interface y Configuración Inicial
Exploración completa de la interfaz de usuario de AutoCAD 2026, incluyendo Application Menu, Quick Access Toolbar, Ribbon, Command Line, y áreas de dibujo. Configuración de espacios de trabajo y personalización del entorno.

### Lección 2: Sistema de Coordenadas y Unidades
Comprensión profunda del sistema de coordenadas cartesianas de AutoCAD, diferencias entre coordenadas absolutas, relativas cartesianas y relativas polares. Configuración de unidades de dibujo y precisión.

### Lección 3: Navegación y Visualización (Zoom, Pan)
Dominio de comandos de navegación: ZOOM (Window, Extents, All, Previous, Realtime), PAN, y navegación con mouse wheel. Uso de vistas guardadas y ViewCube.

### Lección 4: Métodos de Selección de Objetos
Técnicas de selección: clic individual, Window (izquierda a derecha), Crossing (derecha a izquierda), Fence, All, Previous, Last. Configuración de modos de selección.

### Lección 5: Sistema de Entrada Dinámica
Uso de Dynamic Input para entrada de coordenadas y dimensiones directamente en el cursor. Configuración de tooltips dinámicos y entrada de comandos.

### Lección 6: Archivos DWG y Plantillas
Gestión de archivos: NEW, OPEN, SAVE, SAVEAS, CLOSE. Creación de plantillas DWT con configuraciones estándar. Uso de Drawing Utilities y recuperación de archivos.

## Consejos para el Éxito

1. **Practica diariamente**: La fluidez con AutoCAD requiere repetición constante
2. **Usa atajos de teclado**: Memoriza comandos desde el inicio (L para LINE, C para CIRCLE)
3. **Experimenta sin miedo**: AutoCAD tiene UNDO ilimitado (Ctrl+Z)
4. **Mantén archivos organizados**: Crea una estructura de carpetas clara desde el inicio
5. **Toma notas**: Documenta atajos y técnicas que descubras
