# Módulo 1: Introducción a Navisworks 2026

## Descripción del Módulo

Este primer módulo establece las bases fundamentales para trabajar con Autodesk Navisworks 2026, la herramienta líder en la industria AEC para coordinación BIM, revisión de proyectos y detección de interferencias. Los estudiantes aprenderán a dominar la interfaz, importar y gestionar modelos de múltiples disciplinas, y utilizar las herramientas esenciales de navegación, visualización y medición que son la base para todos los flujos de trabajo avanzados de Navisworks.

## Objetivos del Módulo

Al completar este módulo, los estudiantes serán capaces de:

✅ Comprender las diferencias entre Navisworks Manage, Simulate y Freedom, y seleccionar la versión adecuada para cada proyecto

✅ Navegar eficientemente por la interfaz ribbon, scene view, project browser y selection tree de Navisworks 2026

✅ Importar y agregar modelos de múltiples formatos (RVT, DWG, IFC, NWC, etc.) desde diferentes disciplinas

✅ Utilizar todas las herramientas de navegación 3D (walk, fly, orbit, look) para recorrer proyectos de cualquier escala

✅ Crear y gestionar viewpoints guardados para revisiones, presentaciones y documentación

✅ Configurar selection sets y search sets para identificar y aislar elementos específicos en modelos federados complejos

✅ Aplicar estilos de visualización, iluminación, materiales y configuraciones de render en tiempo real

✅ Realizar mediciones precisas punto a punto, distancias mínimas, áreas y ángulos

✅ Utilizar herramientas redline para markup y anotaciones directamente sobre el modelo 3D

## Duración

**6 horas totales** (6 lecciones de 1 hora cada una)

## Estructura de Lecciones

### Lección 1: Interfaz de Navisworks 2026 y Fundamentos
- Diferencias entre versiones (Manage, Simulate, Freedom)
- Tour completo de la interfaz ribbon y paneles
- Navegación 3D básica y controles
- Selection tools y selection tree
- Configuración inicial y preferencias
- Novedades de Navisworks 2026

### Lección 2: Importación y Agregación de Modelos
- Formatos soportados y file readers
- Diferencias entre Append y Merge
- Configuración de unidades y sistemas de coordenadas
- Actualización y refresh de modelos
- Gestión de modelos grandes (large file handling)
- Best practices para federación de modelos

### Lección 3: Navegación y Viewpoints
- Modos de navegación (walk, fly, orbit, look)
- Steering wheels y ViewCube avanzado
- Creación y gestión de saved viewpoints
- Animaciones entre viewpoints
- Section planes y boxing para cortes
- Measure tools básicos

### Lección 4: Selection Sets y Search Sets
- Navegación por selection tree
- Quick properties de objetos
- Creación de search sets con criterios múltiples
- Selection sets para guardar selecciones
- Appearance profiler para control visual
- Operaciones Hide/Required/Override

### Lección 5: Visualización y Apariencia
- Display styles (shaded, realistic, presentation)
- Configuración de iluminación (scene lights, headlamp)
- Aplicación de materiales y texturas
- Render settings para calidad óptima
- Backgrounds (solid, gradient, image)
- Real-time rendering techniques

### Lección 6: Measure y Redline Tools
- Mediciones punto a punto precisas
- Shortest distance entre elementos
- Accumulated measurements
- Medición de áreas y volúmenes
- Medición de ángulos
- Redline markup tools para anotaciones

## Aplicaciones en Proyectos Reales

Este módulo prepara a los estudiantes para:

- **Revisión de modelos federados**: Combinar modelos arquitectónicos, estructurales y MEP en un solo entorno coordinado
- **Presentaciones a clientes**: Crear vistas espectaculares y recorridos virtuales profesionales
- **Validación de diseño**: Verificar dimensiones, espacios y relaciones geométricas antes de construcción
- **Comunicación con equipos**: Crear viewpoints específicos para RFIs, submittals y coordinación
- **Quality control**: Inspección visual detallada de todos los sistemas del proyecto

## Requisitos Previos

- Conocimientos básicos de BIM y procesos de construcción
- Familiaridad con software CAD o BIM (Revit, AutoCAD, ArchiCAD, etc.)
- Comprensión de planos de construcción y coordenadas 3D
- Windows 10 64-bit o superior con GPU compatible DirectX 11

## Software y Hardware

**Software requerido:**
- Autodesk Navisworks Manage 2026 (versión completa, no Freedom)
- Modelos de ejemplo en formatos RVT, DWG, IFC, NWC

**Hardware mínimo:**
- Procesador: Intel Core i5 o superior
- RAM: 16GB mínimo (32GB recomendado para proyectos grandes)
- GPU: Tarjeta gráfica compatible DirectX 11 con 2GB VRAM
- Almacenamiento: 20GB espacio libre
- Monitor: Resolución 1920x1080 mínimo
