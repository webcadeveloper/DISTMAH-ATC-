# Módulo 1: Introducción a Civil 3D 2026

## Descripción del Módulo

El **Módulo 1** introduce la interfaz de Autodesk Civil 3D 2026, workspace especializado para ingeniería civil, configuración de proyectos, estilos (styles), templates, y las nuevas funcionalidades de la versión 2026 incluyendo Project Explorer mejorado y 3D Model Viewer.

Civil 3D es el software líder de Autodesk para diseño de infraestructura civil aplicando metodología BIM (Building Information Modeling) a proyectos de carreteras, urbanizaciones, drenaje, y topografía.

**Duración:** 5 horas
**Lecciones:** 5

---

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

✅ Navegar la interfaz de Civil 3D 2026 y workspace
✅ Utilizar Toolspace (Prospector, Settings, Survey, Toolbox)
✅ Configurar estilos (styles) y templates de proyecto
✅ Establecer unidades, coordenadas y sistemas de referencia
✅ Usar Project Explorer mejorado (novedad 2026)
✅ Visualizar modelos con 3D Model Viewer (nuevo 2026)
✅ Aplicar drawing settings y estándares profesionales
✅ Comprender el flujo de trabajo BIM en infraestructura civil
