# Módulo 6: Documentación y Plan Production

## Descripción del Módulo

El **Módulo 6** cierra el curso de Civil 3D 2026 Básico cubriendo la generación de documentación técnica profesional para proyectos de infraestructura civil. Aprenderás a crear secciones transversales (cross sections), generar vistas de secciones, tablas dinámicas, anotaciones automáticas, y producir planos de construcción completos utilizando Plan Production Tools de Civil 3D.

Este módulo transforma tu modelo 3D en documentación constructiva profesional lista para licitación y construcción, aplicando estándares de la industria y optimizando la productividad con herramientas automatizadas de Civil 3D 2026.

**Duración:** 3 horas
**Lecciones:** 3

---

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

✅ Crear sample lines para secciones transversales de carreteras
✅ Generar section views con información de corte/relleno
✅ Calcular volúmenes de movimiento de tierras por secciones
✅ Crear tablas dinámicas (curves, parcels, volumes)
✅ Aplicar labels y anotaciones automáticas a objetos Civil 3D
✅ Usar Plan Production Tools para generar planos masivamente
✅ Configurar layouts profesionales con viewports
✅ Exportar planos a PDF y DWG siguiendo estándares
✅ Aplicar workflows de plotting y estándares de impresión
✅ Producir sets completos de planos de construcción

## Contenido del Módulo

### [Lección 1: Secciones Transversales y Cálculo de Volúmenes](./lecciones/leccion-1.md)
- ¿Qué son las secciones transversales (cross sections)?
- Sample Lines: creación y configuración
- Sample Line Groups y criterios de muestreo
- Section Views: vistas de secciones transversales
- Estilos de Section Views y personalización
- Cálculo de áreas de corte y relleno
- Volúmenes por secciones (Composite Volumes)
- Total Volume Tables y reportes

**Duración:** 60 minutos

---

### [Lección 2: Sample Lines y Section Views Avanzado](./lecciones/leccion-2.md)
- Sample Lines a lo largo de alineamientos
- Configuración de muestreo (station range, interval)
- Width de sample lines (offset derecho/izquierdo)
- Multiple Section Views: grid layout
- Section View Bands (datos de elevación, offset)
- Hatch patterns para corte/relleno
- Datum elevation y vertical exaggeration
- Proyección de utilities en secciones
- Update automático de secciones al modificar diseño

**Duración:** 60 minutos

---

### [Lección 3: Tables, Labels, Layouts y Exportación Final](./lecciones/leccion-3.md)
- Label Styles para objetos Civil 3D
- Surface labels (elevations, slopes, contours)
- Alignment labels (station/offset, geometry)
- Profile labels (grades, PVIs, curves)
- Table Styles y creación de tablas dinámicas
- Curve Tables, Line Tables, Parcel Tables
- Volume Tables y earthwork reports
- Plan Production Tools (generación masiva de sheets)
- Create Sheets automático por estación
- Layouts profesionales con title blocks
- Page Setup Manager y plot styles
- Batch plotting a PDF
- Exportación a DWG para clientes
- Conclusión del curso Civil 3D 2026 Básico

**Duración:** 60 minutos
