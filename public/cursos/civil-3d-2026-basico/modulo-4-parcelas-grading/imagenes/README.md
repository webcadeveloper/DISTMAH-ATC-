## Formato de Imágenes

- **Formato:** PNG (recomendado) o JPG
- **Resolución:** 1920x1080 o superior para capturas de pantalla principales
- **Resolución mínima:** 1280x720 para capturas secundarias
- **DPI:** 96 DPI (estándar web)
- **Nomenclatura:** Exactamente como se lista arriba (lowercase, guiones)

## Ubicación de Referencias

Las imágenes se referencian en los archivos .md con la ruta relativa:
```markdown

```
