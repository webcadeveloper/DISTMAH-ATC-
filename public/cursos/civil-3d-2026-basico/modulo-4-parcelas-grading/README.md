# Módulo 4: Parcelas y Grading

## Descripción del Módulo

El **Módulo 4** cubre el diseño de parcelas para subdivisiones residenciales y comerciales, junto con técnicas avanzadas de grading (nivelación de terrenos) para plataformas, taludes, y urbanizaciones. Aprenderás a utilizar las herramientas de parcelas de Civil 3D para crear layouts de subdivisión con dimensionamiento automático, gestión de derechos de vía, y feature lines para diseño preciso de grading.

El grading es fundamental en proyectos de urbanización: define cómo fluye el agua superficial, determina los costos de movimiento de tierra (corte y relleno), y establece las plataformas para construcción de edificaciones.

**Duración:** 5 horas
**Lecciones:** 5
