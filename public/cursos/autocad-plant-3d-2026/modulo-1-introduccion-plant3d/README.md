# Módulo 1: Introducción a AutoCAD Plant 3D 2026

## Información del Módulo

- **Duración:** 5 horas
- **Lecciones:** 5
- **Nivel:** Avanzado
- **Prerequisitos:** AutoCAD 2D/3D intermedio

## Descripción

Este módulo introduce los fundamentos esenciales de AutoCAD Plant 3D 2026, la plataforma líder de Autodesk para el diseño de plantas industriales. Aprenderá a configurar proyectos completos, gestionar especificaciones de tuberías con Spec Editor, personalizar propiedades con Data Manager y establecer estándares profesionales siguiendo normas ASME, API e ISO.

AutoCAD Plant 3D 2026 incluye mejoras significativas en rendimiento, interfaz mejorada y nuevas capacidades de colaboración en la nube que optimizan el flujo de trabajo en proyectos industriales complejos.

## Objetivos del Módulo

Al completar este módulo, usted será capaz de:

- ✅ Navegar eficientemente en la interfaz de Plant 3D 2026 y utilizar el Project Manager
- ✅ Crear y configurar proyectos industriales con Plant Project y Plant Settings
- ✅ Gestionar especificaciones de tuberías (Pipe Specs) con el Spec Editor
- ✅ Personalizar propiedades de componentes utilizando Data Manager
- ✅ Establecer estándares de proyecto completos según normas industriales

## Contenido del Módulo

### Lección 1: Interfaz de Plant 3D 2026 y Project Manager
- **Duración:** 60 minutos
- **Archivo:** [leccion-1.md](./lecciones/leccion-1.md)

Exploración completa de la interfaz renovada de Plant 3D 2026, ribbons, paletas de herramientas, Project Manager y configuración inicial del espacio de trabajo para proyectos industriales.

### Lección 2: Configuración de Plant Project y Plant Settings
- **Duración:** 60 minutos
- **Archivo:** [leccion-2.md](./lecciones/leccion-2.md)

Creación de proyectos desde cero, configuración de Plant Settings, gestión de rutas de proyecto, configuración de unidades y sistemas de coordenadas para plantas industriales.

### Lección 3: Spec Editor y Pipeline Specifications
- **Duración:** 60 minutos
- **Archivo:** [leccion-3.md](./lecciones/leccion-3.md)

Dominio del Spec Editor para crear y modificar especificaciones de tuberías (Pipe Specs), selección de materiales, diámetros nominales, ratings de presión y componentes según ASME B31.3.

### Lección 4: Data Manager y Custom Properties
- **Duración:** 60 minutos
- **Archivo:** [leccion-4.md](./lecciones/leccion-4.md)

Utilización de Data Manager para personalizar propiedades de componentes, crear campos personalizados, vincular datos externos y gestionar información de proyecto para reportes.

### Lección 5: Standards y Plant 3D Project Setup completo
- **Duración:** 60 minutos
- **Archivo:** [leccion-5.md](./lecciones/leccion-5.md)

Establecimiento de estándares de proyecto completos, configuración de capas (layers), estilos de texto, dimensionamiento, símbolos y preparación de plantillas reutilizables según normas ISO y ASME.

## Ejercicios Prácticos

⚠️ **Requisitos técnicos:**
- AutoCAD Plant 3D 2026 instalado y activado
- Windows 10/11 64-bit
- 16 GB RAM mínimo (32 GB recomendado)
- Tarjeta gráfica compatible con DirectX 12

- Verifique que tiene permisos de administrador para crear proyectos
- Descargue los archivos de práctica proporcionados
- Configure una carpeta de trabajo en su disco local (no en red)
