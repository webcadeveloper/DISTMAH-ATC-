## Información del Módulo

**Duración:** 5 horas
**Nivel:** Avanzado
**Requisitos previos:** Módulos 1-4 de Civil 3D 2026 Avanzado
**Software requerido:**
- Autodesk Civil 3D 2026

- Visual Studio Code (recomendado para desarrollo)

## Descripción del Módulo

Este módulo te transformará en un experto en automatización de Civil 3D utilizando Dynamo, Python scripting y desarrollo de herramientas personalizadas. Aprenderás a crear flujos de trabajo automatizados que incrementan la productividad hasta 10x en proyectos de infraestructura civil.

Dynamo for Civil 3D es la herramienta de programación visual que permite automatizar tareas repetitivas, crear geometría paramétrica compleja y desarrollar herramientas personalizadas sin necesidad de programación tradicional. Este módulo te enseñará desde los fundamentos hasta técnicas avanzadas de BIM Automation.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

1. **Dominar Dynamo for Civil 3D**
   - Comprender la interfaz y nodos fundamentales
   - Crear scripts visuales para automatización
   - Integrar Dynamo con objetos de Civil 3D
   - Manipular datos entre Excel, Civil 3D y Dynamo

2. **Automatizar Workflows de Diseño**
   - Crear alineamientos automáticos desde datos
   - Generar perfiles y ensamblajes parametricamente
   - Automatizar creación de corredores completos
   - Producir planos automáticamente desde templates

3. **Programar en Python para Civil 3D**
   - Entender la API de Civil 3D con Python
   - Crear scripts para tareas batch
   - Automatizar reportes y extracción de datos
   - Integrar bases de datos externas

4. **Desarrollar Herramientas Personalizadas**
   - Crear custom nodes en Dynamo
   - Desarrollar add-ins para Civil 3D
   - Construir interfaces de usuario simples
   - Distribuir herramientas al equipo

5. **Implementar BIM Automation Avanzada**
   - Automatizar validación de modelos BIM
   - Crear sistemas de nomenclatura inteligente
   - Implementar control de calidad automatizado
   - Generar documentación BIM automática
