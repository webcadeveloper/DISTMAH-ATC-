# MÓDULO 2: INTERSECCIONES Y ROUNDABOUTS

## CIVIL 3D 2026 AVANZADO - DISTMAH ATC

---

## DESCRIPCIÓN DEL MÓDULO

Este módulo profesional cubre el diseño completo de intersecciones viales y roundabouts (glorietas/rotondas) utilizando Civil 3D 2026. Aprenderás a aplicar normas AASHTO (American Association of State Highway and Transportation Officials) para crear intersecciones seguras, eficientes y que cumplan con estándares internacionales de diseño geométrico.

**Nivel:** Avanzado
**Duración estimada:** 10-12 horas
**Prerequisitos:** Civil 3D Básico completado o experiencia equivalente

---

## OBJETIVOS DE APRENDIZAJE

Al finalizar este módulo, serás capaz de:

1. **Diseñar intersecciones** siguiendo normas AASHTO y estándares internacionales
2. **Crear roundabouts completos** con radios de diseño apropiados y análisis de capacidad
3. **Implementar carriles de giro** (turn lanes) con transiciones geométricas correctas
4. **Diseñar carriles de aceleración/desaceleración** para autopistas y vías principales
5. **Utilizar el Create Intersection Wizard** para automatizar diseños complejos
6. **Analizar sight distance** (distancia de visibilidad) y capacidad vehicular
7. **Generar documentación técnica** completa de intersecciones

- Sight distance analysis tools

**Duración:** 2.5 horas

**Contenido:**
- Tipos de roundabouts (mini, single-lane, multi-lane, turbo)
- Normas AASHTO para roundabouts modernos

- Entry/exit radii y deflection angles
- Splitter islands y truck apron design
- Fastest path analysis y speed reduction

- Pedestrian crossings y accessibility (ADA compliance)

**Herramientas Civil 3D:**
- Roundabout wizard
- Swept path analysis
- Vehicle tracking simulation
- Fastest path calculation tools

### LECCIÓN 3: Turn Lanes y Acceleration/Deceleration Lanes (1200+ palabras)
