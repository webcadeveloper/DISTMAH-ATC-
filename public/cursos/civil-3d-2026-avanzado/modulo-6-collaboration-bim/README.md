## Información del Módulo

**Duración:** 4 horas
**Nivel:** Avanzado
**Requisitos previos:** Módulos 1-5 de Civil 3D 2026 Avanzado
**Software requerido:**

- Navegador web moderno (Chrome, Edge)

## Descripción del Módulo

Este módulo final te convierte en experto en colaboración BIM y coordinación de proyectos de infraestructura civil. Aprenderás a trabajar con Autodesk Construction Cloud, BIM 360, Navisworks y formatos de intercambio BIM estándar como IFC y LandXML.

La colaboración efectiva es esencial en proyectos modernos de infraestructura donde múltiples disciplinas (civil, estructural, arquitectónico, MEP) deben coordinarse perfectamente. Este módulo te enseña las mejores prácticas y herramientas para colaboración sin fricción.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

1. **Dominar Autodesk Construction Cloud (ACC)**
   - Configurar proyectos en la nube
   - Gestionar permisos y colaboradores
   - Workflows de Design Collaboration
   - Versionado y control de cambios
   - Revisión y markup de diseños

2. **Integrar BIM 360 en Workflows**
   - Docs y gestión documental
   - Issues y RFIs (Request for Information)
   - Model Coordination
   - Field Management
   - Analytics y reportes

3. **Coordinar Modelos con Navisworks**
   - Federación de modelos multidisciplinarios
   - Clash Detection (detección de interferencias)
   - 4D Construction Simulation
   - Quantification avanzada
   - Revisión virtual de construcción

4. **Exportar a Formatos BIM Estándar**
   - IFC (Industry Foundation Classes)
   - LandXML para intercambio civil
   - CityGML para modelos urbanos
   - Best practices de intercambio

### Proyecto Final Requerido

**Workflow de Colaboración Multidisciplinaria**

Deberás configurar un proyecto completo que incluya:

1. **ACC Project Setup**
   - Crear proyecto en ACC
   - Invitar 3+ colaboradores (roles: diseñador, revisor, gestor)
   - Configurar carpetas y permisos
   - Subir modelos Civil 3D

2. **Design Collaboration**
   - Publicar 3 versiones del diseño
   - Crear markups y comentarios
   - Gestionar issues y resoluciones
   - Sincronizar cambios

3. **Navisworks Coordination**
   - Federar modelo civil con estructural
   - Ejecutar clash detection
   - Documentar 10+ clashes encontrados
   - Proponer soluciones

4. **BIM Exchange**
   - Exportar modelo a IFC 4
   - Exportar alineamientos a LandXML
   - Validar exports con herramientas
   - Documentar proceso

Entregables:
- Proyecto ACC funcional (screenshots)
- Reporte de clash detection
- Archivos IFC y LandXML validados
- Documentación completa del workflow
- Video demo de 10 minutos
