# Módulo 4: Sistemas Especiales y Documentación

## Descripción del Módulo

El **Módulo 4** (módulo final) integra sistemas especiales de plomería como fire protection (sprinklers), gas piping, y medical gases, además de cubrir la documentación técnica profesional completa, dibujos isométricos, coordinación multidisciplinaria MEP mejorada en Revit 2026, y exportación para fabricación. Este módulo cierra el curso con workflows completos desde diseño hasta construcción.

Los sistemas contra incendios son críticos para seguridad de vida (life safety) y requeridos por códigos NFPA. Gas piping provee energía para cocinas, calefacción, y procesos industriales. La documentación profesional y coordinación BIM aseguran proyectos exitosos sin interferencias.

**Duración:** 4 horas
**Lecciones:** 4

---

## Códigos y Normas Aplicados

**Fire Protection:**
- **NFPA 13:** Installation of Sprinkler Systems
- **NFPA 14:** Standpipe and Hose Systems
- **NFPA 20:** Fire Pumps
- **NFPA 24:** Private Fire Service Mains

**Gas Piping:**
- **NFPA 54 (ANSI Z223.1):** National Fuel Gas Code
- **IFGC:** International Fuel Gas Code
- **NFPA 58:** LP-Gas Code

**Medical Gases:**
- **NFPA 99:** Health Care Facilities Code
- **ASME A13.1:** Pipe Marking (color codes)

**BIM/Coordination:**
- **NBIMS (National BIM Standard):** US BIM guidelines
- **ISO 19650:** International BIM standards
