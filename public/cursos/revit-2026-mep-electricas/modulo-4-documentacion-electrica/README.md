# Módulo 4: Documentación Eléctrica

## Descripción General

El Módulo 4 de MEP Eléctricas en Revit 2026 se enfoca en la generación, gestión y coordinación de documentación técnica eléctrica profesional. Este módulo es fundamental para transformar modelos BIM en conjuntos de planos constructivos completos, precisos y coordinados que cumplan con normativas internacionales.

## Objetivos de Aprendizaje

Al completar este módulo, los estudiantes serán capaces de:

- Crear y gestionar schedules de paneles y circuitos eléctricos
- Generar diagramas unifilares precisos y profesionales
- Implementar sistemas de detección de conflictos (clash detection)
- Coordinar disciplinas MEP para evitar interferencias
- Documentar instalaciones eléctricas según estándares internacionales
- Exportar y organizar planos para presentación a autoridades

## Contenido del Módulo

### Lección 1: Panel Schedules y Circuit Schedules
Aprender a crear tablas de paneles y circuitos, configurar parámetros, y generar documentación automática que refleje fielmente el modelo.

### Lección 2: Diagramas Unifilares
Desarrollar habilidades para crear diagramas esquemáticos unifilares que comuniquen de manera clara la topología de la instalación eléctrica.

### Lección 3: Coordinación y Clash Detection
Implementar metodologías de coordinación interdisciplinaria y detección automática de conflictos para garantizar constructibilidad.

## Requisitos Previos

- Completar Módulos 1, 2 y 3
- Conocimiento de Revit MEP nivel intermedio
- Comprensión de sistemas eléctricos básicos
- Familiaridad con normativas de instalaciones eléctricas

## Duración Estimada

**40 horas** de formación teórica y práctica

## Metodología

Este módulo combina:
- **Lecciones teóricas**: Conceptos y mejores prácticas profesionales
- **Ejercicios prácticos**: Aplicación real con modelos de casos de estudio
- **Proyectos integradores**: Documentación completa de instalaciones reales
- **Revisión de estándares**: Normativas IEC, NFPA, NEC y locales
