# Módulo 1: Fundamentos Eléctricos en Revit MEP

## Descripción

Este módulo introduce los conceptos fundamentales para el diseño de instalaciones eléctricas en Revit MEP 2026. Aprenderás a configurar proyectos eléctricos desde cero, dominar los electrical settings, implementar las nuevas características de conductores personalizables y comprender los sistemas de voltaje y distribución eléctrica.

**Duración:** 4 horas
**Lecciones:** 4
**Nivel:** Intermedio

---

## Objetivos del Módulo

Al completar este módulo, serás capaz de:

1. Navegar eficientemente en la interface MEP especializada para diseño eléctrico
2. Configurar electrical settings profesionales para proyectos complejos
3. Implementar conductores y cables personalizables utilizando las nuevas características de Revit 2026
4. Definir materiales, diámetros y tipos de aislamiento para conductores
5. Configurar sistemas de voltaje (120V, 208V, 277V, 480V) correctamente
6. Diseñar sistemas de distribución eléctrica trifásicos
7. Cargar y utilizar familias eléctricas profesionales
8. Navegar y organizar modelos eléctricos complejos

---

## Contenido del Módulo

### Lección 1: Interface MEP y Configuración de Proyecto Eléctrico
- Interface especializada de Revit MEP para sistemas eléctricos
- Configuración inicial de proyectos eléctricos
- Electrical Settings: voltajes, distribución y sistemas
- Unidades eléctricas y estándares internacionales
- Plantillas de proyecto eléctrico profesionales

### Lección 2: Conductores y Cables Personalizables en Revit 2026
- NUEVO 2026: Sistema de conductores personalizables
- Definición de materiales conductores (cobre, aluminio)
- Configuración de diámetros y calibres (AWG, IEC, métrico)
- Tipos de aislamiento y temperatura de operación
- Propiedades físicas y eléctricas de conductores
- Bibliotecas de conductores por normativa

### Lección 3: Sistemas de Voltaje y Distribución Eléctrica
- Sistemas de voltaje monofásicos (120V, 240V)
- Sistemas de voltaje trifásicos (208V, 277V, 480V)
- Configuraciones Delta y Estrella (Wye)
- Sistemas de distribución: TN, TT, IT
- Tierra y neutro en sistemas eléctricos
- Voltage drop y caída de tensión

### Lección 4: Familias Eléctricas y Navegación en Modelos
- Categorías de familias eléctricas en Revit
- Familias de tableros, equipos y dispositivos
- Parámetros eléctricos en familias
- Navegación en el Project Browser para sistemas eléctricos
- Vistas especializadas para diseño eléctrico
- Filtros y visibilidad de elementos eléctricos

## Novedades Revit 2026 en Este Módulo

- **Conductores Personalizables:** Definición completa de propiedades de conductores
- **Materiales Conductores:** Biblioteca expandida de materiales con propiedades físicas
