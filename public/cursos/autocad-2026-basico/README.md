# AutoCAD 2026 Básico

**Duración Total:** 30 horas

---

## Descripción del Curso

Este curso de AutoCAD 2026 Básico está diseñado para proporcionar a los estudiantes una base sólida en las técnicas fundamentales de diseño asistido por computadora (CAD). Los participantes aprenderán a crear, editar y gestionar dibujos técnicos 2D utilizando las herramientas más recientes de AutoCAD 2026.

---

## Público Objetivo

- Principiantes sin experiencia previa en AutoCAD
- Estudiantes de ingeniería, arquitectura y diseño
- Profesionales que desean iniciar su transición al diseño CAD
- Técnicos que necesitan habilidades básicas de dibujo técnico

---

## Prerrequisitos

- Conocimientos básicos de computación
- Familiaridad con el sistema operativo Windows
- Comprensión básica de geometría y dibujo técnico (recomendado)
- Computadora con AutoCAD 2026 instalado

---

## Objetivos de Aprendizaje

Al finalizar este curso, los estudiantes serán capaces de:

1. Navegar eficientemente en la interfaz de AutoCAD 2026
2. Crear y gestionar dibujos técnicos 2D precisos
3. Utilizar capas para organizar información de diseño
4. Aplicar comandos básicos de dibujo y edición
5. Implementar Object Snap para precisión en el dibujo
6. Crear anotaciones y dimensiones básicas
7. Configurar y guardar archivos de dibujo

---

## Estructura del Curso

### **Módulo 1: Introducción a AutoCAD 2026** (4-5 horas)

#### Temas:
- **1.1 Interfaz de Usuario**
  - Ribbon, Panels y Tabs
  - Command Line y Dynamic Input
  - Workspace personalizado
  - Quick Access Toolbar

- **1.2 Configuración Inicial**
  - Unidades de dibujo (Metric/Imperial)
  - Límites del dibujo (Drawing Limits)
  - Sistema de coordenadas (UCS)
  - Grid y Snap Mode

- **1.3 Gestión de Archivos**
  - Crear nuevo dibujo (New)
  - Abrir archivos existentes (Open)
  - Guardar y Guardar Como (Save/Save As)
  - Formatos de archivo (.dwg, .dxf)

- **1.4 Navegación Básica**
  - Zoom básico (Window, Extents, All)
  - Pan para desplazamiento
  - Uso del scroll wheel del mouse
  - ViewCube y Navigation Bar

#### Ejercicios Prácticos:
- Configurar un nuevo proyecto de dibujo
- Personalizar el workspace
- Navegar en dibujos existentes

---

### **Módulo 2: Construcción Básica** (8-10 horas) ✅ YA CREADO

#### Temas:
- **2.1 Capas (Layers)**
  - Crear y gestionar capas
  - Propiedades de capas (Color, Linetype, Lineweight)
  - Layer States (On/Off, Freeze/Thaw, Lock/Unlock)
  - Layer Properties Manager
  - ByLayer vs ByBlock

- **2.2 Comando Arc**
  - Arc 3 Points
  - Arc Start, Center, End
  - Arc Start, End, Angle
  - Arc Start, Center, Length
  - Arc Continue

- **2.3 Comando Circle**
  - Circle Center, Radius
  - Circle Center, Diameter
  - Circle 2-Point
  - Circle 3-Point
  - Circle Tan, Tan, Radius
  - Circle Tan, Tan, Tan

- **2.4 Object Snap (OSNAP)**
  - Running Object Snap vs Override
  - Endpoint, Midpoint, Center
  - Node, Quadrant, Intersection
  - Tangent, Perpendicular, Nearest
  - AutoSnap markers y tooltips

- **2.5 Zoom y Pan Avanzado**
  - Zoom Window, Previous, Realtime
  - Zoom In/Out (Scale factors)
  - Zoom Dynamic
  - Pan Realtime
  - Scroll wheel techniques

- **2.6 Comando Ellipse**
  - Ellipse Axis, End
  - Ellipse Center
  - Elliptical Arc
  - Rotation method

#### Ejercicios Prácticos:
- Dibujar subdivision plan con arcos tangentes
- Crear gaskets y flanges con círculos
- Practicar Object Snap en dibujos complejos
- Ejercicios de navegación eficiente

---

### **Módulo 3: Comandos de Dibujo Fundamentales** (6-7 horas)

#### Temas:
- **3.1 Comando Line**
  - Absolute coordinates (X,Y)
  - Relative coordinates (@X,Y)
  - Polar coordinates (@distance<angle)
  - Direct distance entry
  - Ortho Mode y Polar Tracking

- **3.2 Comando Polyline (PLINE)**
  - Polyline vs Line
  - Arc mode en Polyline
  - Width y Halfwidth
  - Cerrar polylines (Close)
  - Edit Polyline (PEDIT básico)

- **3.3 Comando Rectangle**
  - Rectangle por esquinas
  - Rectangle con dimensiones
  - Chamfer y Fillet options
  - Rotation option
  - Area option

- **3.4 Comando Polygon**
  - Inscribed vs Circumscribed
  - Edge method
  - Número de lados (3-1024)
  - Center point method

- **3.5 Comando Spline**
  - Fit points vs Control vertices
  - Spline de-a través de puntos
  - Close spline
  - Edit Spline (SPLINEDIT básico)

#### Ejercicios Prácticos:
- Crear plot plans con líneas y coordenadas
- Dibujar componentes mecánicos con polylines
- Diseñar formas arquitectónicas con rectángulos
- Crear símbolos técnicos con polígonos

---

### **Módulo 4: Edición Básica** (6-7 horas)

#### Temas:
- **4.1 Selección de Objetos**
  - Window vs Crossing
  - Selection modes
  - Selection Cycling
  - Quick Select
  - Select Similar

- **4.2 Comandos de Eliminación y Modificación**
  - Erase (Delete)
  - Undo/Redo
  - Oops command

- **4.3 Comandos de Transformación**
  - Move (desplazar objetos)
  - Copy (copiar objetos)
  - Rotate (rotar objetos)
  - Scale (escalar objetos)

- **4.4 Comandos de Simetría y Alineación**
  - Mirror (espejo)
  - Mirror con/sin borrar original
  - Mirror Text (MIRRTEXT variable)

- **4.5 Comandos de Recorte y Extensión**
  - Trim (recortar)
  - Extend (extender)
  - Edge mode y Project mode
  - Fence y Crossing selections

#### Ejercicios Prácticos:
- Modificar drawings existentes
- Crear patrones simétricos con Mirror
- Ajustar dimensiones con Scale
- Limpiar dibujos con Trim y Extend

---

### **Módulo 5: Anotaciones Básicas** (5-6 horas)

#### Temas:
- **5.1 Single Line Text (DTEXT)**
  - Justify options
  - Height y Rotation
  - Text Styles
  - Special characters

- **5.2 Multiline Text (MTEXT)**
  - Text Editor interface
  - Formatting options
  - Bullets y Numbering
  - Columns
  - Import text

- **5.3 Dimensiones Básicas**
  - Linear dimensions
  - Aligned dimensions
  - Angular dimensions
  - Radius y Diameter dimensions
  - Dimension Styles básico

- **5.4 Leaders Básicos**
  - Quick Leader
  - Multileader (MLEADER)
  - Leader Styles
  - Annotative scaling introduction

#### Ejercicios Prácticos:
- Anotar plot plans
- Dimensionar componentes mecánicos
- Crear notas técnicas
- Añadir títulos y labels
