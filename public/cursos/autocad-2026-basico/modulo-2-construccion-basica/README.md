# Módulo 2: Técnicas de Construcción Básica en AutoCAD 2026

## Objetivos del Módulo

Al completar este módulo, usted será capaz de:

1. Abrir dibujos existentes
2. Trabajar con capas nuevas y existentes
3. Dibujar usando los comandos Arc y Circle
4. Configurar y usar Object Snaps en ejecución
5. Cambiar la visualización usando Zoom y Pan
6. Usar Dynamic View
7. Dibujar elipses

## Contenido del Módulo

### Lección 1: Introducción y Apertura de Dibujos
- Abrir dibujos existentes
- Guardar como nuevo archivo
- Usar la herramienta Find para buscar archivos

### Lección 2: Trabajando con Capas (Layers)
- Conceptos de capas en AutoCAD
- Crear nuevas capas
- Controlar visibilidad de capas
- Congelar y bloquear capas
- Control de colores y tipos de línea

### Lección 3: Comando Arc
- Arc 3 Points
- Arc Start, Center, End
- Arc Start, End, Angle
- Arc Start, Center, Length
- Arc Start, End, Radius
- Continuar arcos

### Lección 4: Comando Circle
- Circle 2-Point
- Circle 3-Point
- Circle Tangent, Tangent, Radius
- Circle Tangent, Tangent, Tangent

### Lección 5: Object Snap
- Configurar Object Snap
- Running mode vs Override mode
- Snap to Node
- Snap to Endpoint
- Snap to Tangent

### Lección 6: Comandos de Visualización
- Zoom Window
- Zoom Previous
- Zoom Realtime
- Zoom All
- Zoom Dynamic
- Pan Realtime
- Uso del scroll wheel

### Lección 7: Comando Ellipse
- Ellipse Axis, End
- Ellipse Center
- Ellipse con rotación
