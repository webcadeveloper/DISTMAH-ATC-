# Módulo 1: Fundamentos de Modelado 3D

## Descripción del Módulo

Este módulo introductorio establece las bases fundamentales para el trabajo tridimensional en AutoCAD 2026. Los estudiantes aprenderán a navegar y configurar el entorno de trabajo 3D, comprender el sistema de coordenadas universales (UCS), dominar las herramientas de visualización tridimensional y crear objetos 3D básicos.

El módulo está diseñado para profesionales que dominan AutoCAD 2D y desean dar el salto al modelado tridimensional, proporcionando una transición estructurada desde el dibujo bidimensional hacia el espacio tridimensional.

## Objetivos de Aprendizaje

Al finalizar este módulo, serás capaz de:

- Configurar y personalizar el workspace 3D Modeling en AutoCAD 2026
- Comprender y manipular el sistema de coordenadas UCS (User Coordinate System)
- Navegar eficientemente en el espacio tridimensional usando ViewCube, Orbit y vistas predefinidas
- Introducir coordenadas 3D mediante diferentes métodos de entrada
- Crear objetos tridimensionales básicos (Box, Cylinder, Sphere, Cone, Pyramid)
- Utilizar herramientas Polysolid y 3D Polyline para modelado lineal
- Generar objetos mesh básicos y comprender sus aplicaciones

## Duración

6 horas distribuidas en 6 lecciones de 1 hora cada una.

## Requisitos Previos

- Conocimiento sólido de AutoCAD 2D (comandos básicos de dibujo y edición)
- Comprensión de sistemas de coordenadas cartesianas
- AutoCAD 2026 instalado y funcional
- Nociones básicas de geometría espacial

## Contenido del Módulo

### Lección 1: Workspace 3D y Sistema UCS
**Duración:** 1 hora

Introducción al entorno de trabajo tridimensional de AutoCAD 2026. Configuración del workspace 3D Modeling, personalización de la interfaz para modelado 3D, y comprensión profunda del sistema de coordenadas UCS (User Coordinate System).

**Temas clave:**
- Activación y configuración del workspace 3D Modeling
- Diferencias entre workspace 2D y 3D
- Concepto y funcionamiento del UCS
- Comandos UCS, PLAN, y UCS Icon
- Sistemas de coordenadas guardados (Named UCS)

### Lección 2: Visualización 3D (ViewCube, Orbit, Views)
**Duración:** 1 hora

Dominio de las herramientas de navegación y visualización tridimensional. Uso del ViewCube, diferentes tipos de Orbit, vistas predefinidas y técnicas de zoom en 3D.

**Temas clave:**
- ViewCube: navegación intuitiva en 3D
- 3D Orbit, Free Orbit, Continuous Orbit
- Vistas isométricas y ortogonales predefinidas
- Comandos VIEW y VPOINT
- Configuración de proyecciones paralelas y perspectivas

### Lección 3: Coordenadas 3D y Métodos de Entrada
**Duración:** 1 hora

Técnicas para introducir coordenadas tridimensionales con precisión. Sistemas de coordenadas absolutas, relativas, cilíndricas y esféricas en el espacio 3D.

**Temas clave:**
- Coordenadas absolutas 3D (X,Y,Z)
- Coordenadas relativas 3D (@X,Y,Z)
- Coordenadas cilíndricas (@distancia<ángulo,Z)
- Coordenadas esféricas (@distancia<ángulo<ángulo)
- Filtros de coordenadas .X, .Y, .Z, .XY

### Lección 4: Objetos 3D Básicos (Box, Cylinder, Sphere)
**Duración:** 1 hora

Creación de primitivas 3D fundamentales. Modelado de cajas, cilindros, esferas, conos, pirámides y toroides con opciones paramétricas.

**Temas clave:**
- Comando BOX: opciones Center, Cube, Length
- Comando CYLINDER: opciones Center, Elliptical
- Comando SPHERE: opciones Center, 3P, 2P, Ttr
- Comando CONE y PYRAMID
- Comando TORUS
- Propiedades paramétricas de primitivas 3D

### Lección 5: Polysolid y 3D Polyline
**Duración:** 1 hora

Herramientas para modelado lineal tridimensional. Creación de muros y estructuras con Polysolid, y trazado de líneas tridimensionales con 3D Polyline.

**Temas clave:**
- Comando POLYSOLID: creación de muros 3D
- Configuración de altura y ancho de Polysolid
- Conversión de líneas 2D a Polysolid
- Comando 3DPOLY: polilíneas tridimensionales
- Edición de vértices en 3D
- Aplicaciones arquitectónicas de Polysolid

### Lección 6: Mesh Básico
**Duración:** 1 hora

Introducción al modelado de mallas (mesh). Creación de objetos mesh básicos, diferencias con sólidos, y técnicas de suavizado.

**Temas clave:**
- Conceptos fundamentales de mesh modeling
- Primitivas mesh: Box, Cylinder, Sphere, etc.
- Comando MESH y opciones de suavizado
- Diferencias entre mesh y sólidos 3D
- Conversión entre mesh y sólidos
- Nivel de smoothness y facetas

## Consejos para el Éxito

1. **Practica la navegación 3D** hasta que sea intuitiva antes de modelar
2. **Domina el UCS** completamente, es la base de todo trabajo 3D
3. **Usa vistas múltiples** (viewports) para verificar tu trabajo desde diferentes ángulos
4. **Guarda UCS personalizados** para planos de trabajo frecuentes
5. **Experimenta con diferentes métodos** de entrada de coordenadas
6. **Verifica dimensiones** constantemente con comandos de medición 3D

## Preparación para el Módulo 2

- Navegación fluida en 3D
- Manipulación del UCS
- Creación precisa de primitivas 3D
- Entrada de coordenadas tridimensionales
