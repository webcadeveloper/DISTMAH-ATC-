# Módulo 2: Modelado de Sólidos 3D

## Descripción del Módulo

El Módulo 2 lleva tus habilidades de modelado 3D al siguiente nivel, introduciendo operaciones avanzadas que transforman primitivas básicas en modelos complejos y profesionales. Aprenderás técnicas de extrusión, revolución, barrido (sweep), loft, operaciones booleanas, y modificaciones de sólidos que son fundamentales en diseño arquitectónico, mecánico e industrial.

Este módulo cubre las herramientas más poderosas de AutoCAD 2026 para modelado de sólidos: EXTRUDE (extrusión de perfiles 2D), REVOLVE (revolución alrededor de eje), SWEEP (barrido a lo largo de trayectoria), LOFT (interpolación entre perfiles), operaciones booleanas (UNION, SUBTRACT, INTERSECT), modificaciones (SHELL, FILLET3D, CHAMFER3D), arrays 3D, mirrors 3D, y herramientas de sección (SECTION, SLICE).

## Objetivos de Aprendizaje

Al finalizar este módulo, serás capaz de:

- Extruir perfiles 2D para crear sólidos complejos con taper y trayectorias
- Revolucionar perfiles alrededor de ejes para crear sólidos de revolución
- Crear sólidos mediante barrido (SWEEP) de perfiles a lo largo de trayectorias
- Generar formas orgánicas con LOFT interpolando entre múltiples perfiles
- Aplicar operaciones booleanas (UNION, SUBTRACT, INTERSECT) para modelado constructivo
- Modificar sólidos con SHELL (ahuecado), FILLET3D (redondeo), CHAMFER3D (chaflán)
- Crear patrones circulares y rectangulares con 3DARRAY
- Duplicar sólidos simétricamente con MIRROR3D
- Analizar modelos con secciones (SECTION) y cortes (SLICE)

## Duración

6 horas distribuidas en 6 lecciones de 1 hora cada una.

## Contenido del Módulo

### Lección 7: Extrude y Presspull
**Duración:** 1 hora

Dominio de la extrusión, la operación más fundamental del modelado 3D. Aprenderás a convertir perfiles 2D (líneas cerradas, polilíneas, círculos, regiones) en sólidos tridimensionales mediante extrusión lineal, con taper (cónico), y a lo largo de trayectorias. PRESSPULL permite extrusión directa mediante clic, ideal para modelado rápido.

**Temas clave:**
- Comando EXTRUDE: extrusión básica, taper, y path (trayectoria)
- Selección de perfiles válidos para extrusión
- Comando PRESSPULL: extrusión dinámica interactiva
- Diferencias entre EXTRUDE y PRESSPULL
- Aplicaciones arquitectónicas y mecánicas

### Lección 8: Revolve y Sweep
**Duración:** 1 hora

Creación de sólidos de revolución (REVOLVE) y sólidos de barrido (SWEEP). REVOLVE rota un perfil alrededor de un eje para crear objetos simétricos radiales. SWEEP arrastra un perfil a lo largo de una trayectoria 3D para crear formas complejas.

**Temas clave:**
- Comando REVOLVE: eje de revolución, ángulos parciales
- Perfiles y ejes válidos para revolución
- Comando SWEEP: perfil y trayectoria (path)
- Opciones de alineación y escala en SWEEP
- Aplicaciones: tornería, tuberías, molduras

### Lección 9: Loft y Operaciones Booleanas
**Duración:** 1 hora

Modelado orgánico con LOFT (transiciones suaves entre perfiles) y modelado constructivo con operaciones booleanas (UNION, SUBTRACT, INTERSECT). Estas técnicas combinadas permiten crear virtualmente cualquier geometría sólida.

**Temas clave:**
- Comando LOFT: interpolación entre múltiples secciones
- Guías (guides) y trayectorias en LOFT
- Comando UNION: unir sólidos
- Comando SUBTRACT: restar sólidos (aberturas, huecos)
- Comando INTERSECT: volumen común
- Estrategias de modelado constructivo

### Lección 10: Shell, Fillet 3D, Chamfer 3D
**Duración:** 1 hora

Modificaciones avanzadas de sólidos. SHELL ahúeca sólidos dejando paredes de grosor especificado. FILLET3D redondea aristas con radios variables. CHAMFER3D crea chaflanes en aristas. Estas operaciones son fundamentales en diseño mecánico e industrial.

**Temas clave:**
- Comando SHELL: ahuecado de sólidos, grosor de pared
- Comando FILLET3D: redondeo de aristas, cadenas de aristas
- Comando CHAMFER3D: chaflanes de distancia y ángulo
- Edición de fillets y chamfers existentes
- Aplicaciones en diseño de piezas mecánicas

### Lección 11: 3D Array y 3D Mirror
**Duración:** 1 hora

Duplicación eficiente de geometría 3D mediante arrays (patrones rectangulares y polares) y mirrors (simetrías). Estas herramientas aceleran dramáticamente el modelado de estructuras repetitivas y simétricas.

**Temas clave:**
- Comando 3DARRAY: arrays rectangulares y polares en 3D
- Especificación de ejes de rotación para arrays polares
- Comando MIRROR3D: simetría respecto a planos 3D
- Definición de planos de espejo (3 puntos, objeto, eje)
- Workflow para estructuras simétricas

### Lección 12: Sección y Slice
**Duración:** 1 hora

Análisis y documentación de modelos 3D mediante secciones y cortes. SECTION crea regiones 2D en planos de corte. SLICE divide sólidos en partes. Estas herramientas son esenciales para documentación técnica y análisis de diseño.

**Temas clave:**
- Comando SECTION: creación de secciones 2D
- Planos de sección (3 puntos, objeto, eje, vista)
- Comando SLICE: corte de sólidos
- Opciones de retención (mantener ambas partes o una)
- Generación de vistas de sección para planos constructivos
- SECTIONPLANE: planos de corte dinámicos

## Consejos para el Éxito

1. **Planifica antes de modelar**: Dibuja perfiles 2D completos antes de extruir o revolucionar
2. **Usa regiones para perfiles complejos**: REGION crea áreas cerradas válidas para extrusión
3. **Verifica dirección de extrusión/revolución**: Visualiza en isométrica antes de ejecutar
4. **Aplica operaciones booleanas en orden lógico**: UNION primero, SUBTRACT al final
5. **Copia antes de operaciones destructivas**: SUBTRACT y SLICE eliminan geometría
6. **Usa historia de construcción**: Algunos comandos permiten editar parámetros después
7. **Verifica geometría válida**: Sólidos deben ser cerrados (watertight) para operaciones booleanas

Los comandos de este módulo se aplican directamente en:

**Arquitectura:**
- Molduras extruidas a lo largo de perímetros

**Diseño Mecánico:**

**Diseño Industrial:**
- Botellas y envases (REVOLVE + SHELL)
- Tuberías complejas (SWEEP)
- Transiciones de sección (LOFT)
- Componentes simétricos (MIRROR3D)

## Flujo de Trabajo Recomendado

### Para Piezas Mecánicas:
1. Dibujar perfil 2D principal (sección transversal)
2. REVOLVE o EXTRUDE para crear forma base
3. Agregar features con primitivas (BOX, CYLINDER)
4. UNION para unir componentes principales
5. SUBTRACT para agujeros y aberturas
6. FILLET3D para redondeos
7. CHAMFER3D para acabados de aristas
8. 3DARRAY para patrones repetitivos

### Para Elementos Arquitectónicos:
1. Dibujar planta o perfil 2D con PLINE
2. EXTRUDE o POLYSOLID para muros/estructura
3. Crear primitivas para elementos adicionales
4. SUBTRACT para puertas, ventanas, aberturas
5. SWEEP para molduras y elementos lineales
6. UNION para consolidar geometría
7. SECTION para vistas de corte (documentación)
