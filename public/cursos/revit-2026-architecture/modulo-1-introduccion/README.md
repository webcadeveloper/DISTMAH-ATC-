## Descripcion del Modulo

Este modulo introductorio te familiarizara con Autodesk Revit Architecture 2026, su interfaz renovada, las nuevas herramientas de navegacion con **Accelerated Graphics** (Tech Preview), y los conceptos fundamentales del modelado BIM. Aprenderas a configurar proyectos, trabajar con plantillas, y dominar el sistema de vistas que es esencial para el trabajo en Revit.

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Navegar fluidamente por la interfaz de Revit 2026
- Utilizar Accelerated Graphics para navegacion 3D mejorada
- Configurar proyectos nuevos con plantillas personalizadas
- Trabajar con el sistema de vistas (plantas, cortes, elevaciones, 3D)
- Crear y modificar niveles y grillas
- Utilizar herramientas basicas de modelado
- Comprender los principios fundamentales de BIM

## Notas Importantes

### Nueva Caracteristica 2026: Accelerated Graphics

La funcion **Accelerated Graphics** es una vista previa tecnologica en Revit 2026 que utiliza la GPU para mejorar significativamente el rendimiento de navegacion 3D.

**Beneficios:**
- Navegacion mas fluida en modelos grandes
- Menor tiempo de respuesta en vistas 3D
- Mejor rendimiento con tarjetas graficas modernas

**Activacion:**
1. Ir a `Options > Graphics`
2. Habilitar "Use Accelerated Graphics (Tech Preview)"
3. Reiniciar Revit

**Requisitos:**
- Tarjeta grafica compatible DirectX 11+
- Drivers actualizados
- Windows 10/11
