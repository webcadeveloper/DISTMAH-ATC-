# Módulo 5: Características Avanzadas y AI de AutoCAD 2026

## Descripción General

Este módulo cubre las características más innovadoras y avanzadas de AutoCAD 2026, incluyendo las nuevas funcionalidades impulsadas por Inteligencia Artificial de Autodesk, herramientas de colaboración en la nube, y mejoras de rendimiento que hacen de esta versión la más rápida hasta la fecha.

**Duración:** 7-8 horas
**Nivel:** Avanzado
**Requisitos previos:** Módulos 1-4 completados

---

## ⚠️ NOTA IMPORTANTE SOBRE CARACTERÍSTICAS 2026

Las características presentadas en este módulo son específicas de AutoCAD 2026 y pueden evolucionar con actualizaciones de software. Los profesores deben:

- ✅ Verificar la disponibilidad actual de características en la última versión de AutoCAD
- ✅ Confirmar requisitos de suscripción (algunas características requieren planes específicos)
- ✅ Revisar la documentación oficial de Autodesk para cambios recientes
- ✅ Adaptar el contenido según las actualizaciones del software
- ✅ Verificar que los estudiantes tengan acceso a las características (Tech Preview, etc.)

**Algunas características están en Tech Preview y pueden cambiar o requerir activación especial.**

---

## Objetivos de Aprendizaje

Al finalizar este módulo, los estudiantes serán capaces de:

1. **Utilizar Smart Blocks con IA** para optimizar dibujos automáticamente
2. **Implementar Connected Support Files** para proyectos colaborativos
3. **Analizar Activity Insights** para seguimiento de cambios y productividad
4. **Integrar Markup Import/Assist** para flujos de revisión eficientes
5. **Aprovechar Autodesk Assistant** para asistencia con IA conversacional
6. **Incorporar ArcGIS Basemaps** en proyectos de infraestructura
7. **Dominar las mejoras de Hatch** para diseños más flexibles
8. **Maximizar la productividad** con las características de 2026

## Contenido del Módulo

### Lección 1: Smart Blocks - Detect and Convert
**Duración:** 50-60 minutos

Aprende a utilizar la detección automática de objetos repetitivos y su conversión inteligente a bloques mediante Autodesk AI.

**Temas:**
- Introducción a Smart Blocks (Tech Preview)
- Comando SMARTBLOCKSDETECT
- Reconocimiento automático de objetos
- Algoritmos de IA para detección
- Flujos de trabajo de optimización
- Mejores prácticas y limitaciones

**Archivo:** [leccion-01-smart-blocks-detect-convert.md](./leccion-01-smart-blocks-detect-convert.md)

---

### Lección 2: Smart Blocks - Search and Convert
**Duración:** 50-60 minutos

Domina la búsqueda inteligente de objetos similares y su conversión a bloques existentes o sugeridos por IA.

**Temas:**
- Comando SMARTBLOCKSSEARCH
- Búsqueda de objetos similares
- Sugerencias de bloques por IA
- Integración con bibliotecas de bloques
- Conversión en lote
- Criterios de similitud

**Archivo:** [leccion-02-smart-blocks-search-convert.md](./leccion-02-smart-blocks-search-convert.md)

---

### Lección 3: Smart Blocks - Replacement y Placement
**Duración:** 50-60 minutos

Aprende a reemplazar bloques inteligentemente y colocar nuevos bloques basándose en patrones detectados por IA.

**Temas:**
- Comando SMARTBLOCKSREPLACE
- Sugerencias automáticas de reemplazo
- Comando SMARTBLOCKSPLACEMENT
- Colocación automática basada en patrones
- Aprendizaje de ubicaciones anteriores
- Estadísticas de productividad

**Archivo:** [leccion-03-smart-blocks-replacement-placement.md](./leccion-03-smart-blocks-replacement-placement.md)

---

### Lección 4: Connected Support Files
**Duración:** 45-55 minutos

Configura archivos de soporte conectados para proyectos colaborativos sin configuración individual por usuario.

**Temas:**
- Introducción a Connected Support Files
- Integración con Autodesk Docs
- Configuración centralizada de proyectos
- Sincronización en la nube
- Beneficios de colaboración
- Gestión de rutas de soporte
- Control de versiones

**Archivo:** [leccion-04-connected-support-files.md](./leccion-04-connected-support-files.md)

---

### Lección 5: Activity Insights - "What's Changed"
**Duración:** 45-55 minutos

Analiza el historial completo de cambios por sesión y usuario para tomar decisiones informadas.

**Temas:**
- Resumen de cambios por sesión
- Seguimiento de objetos modificados
- Análisis de flujos de trabajo
- Tiempo total de edición
- Seguimiento multi-usuario
- Auditoría de dibujos
- Aplicaciones en gestión de proyectos

**Archivo:** [leccion-05-activity-insights.md](./leccion-05-activity-insights.md)

---

### Lección 6: Markup Import and Markup Assist
**Duración:** 50-60 minutos

Importa y gestiona marcas desde Autodesk Docs con sincronización bidireccional y asistencia para incorporar cambios.

**Temas:**
- Importación de markups JPG, PNG, PDF
- Sincronización con Autodesk Docs
- Markup Assist para incorporar cambios
- Integración con Autodesk Docs Issues
- Flujos de revisión y aprobación
- Colaboración con stakeholders
- Optimización de ciclos de revisión

**Archivo:** [leccion-06-markup-import-assist.md](./leccion-06-markup-import-assist.md)

### Lección 7: Autodesk Assistant - IA Conversacional
**Duración:** 40-50 minutos

Utiliza el asistente conversacional de IA para obtener ayuda contextual, guías generadas y recomendaciones personalizadas.

**Temas:**
- Interfaz de IA conversacional
- Preguntas en lenguaje natural
- Guías y resúmenes generados
- Recomendaciones de recursos
- Asistencia contextual
- Resolución de problemas en tiempo real
- Descubrimiento de características

## Requisitos del Sistema

### Software
- **AutoCAD 2026** (versión completa con suscripción activa)
- **Acceso a Autodesk Docs** (para Connected Support Files y Markup Import)
- **Conexión a Internet** (para características de IA y sincronización en la nube)

### Hardware Recomendado
- **Procesador:** Multi-core 3+ GHz
- **RAM:** 16 GB mínimo, 32 GB recomendado
- **GPU:** GPU dedicada con 4+ GB VRAM (para rendimiento óptimo de IA)
- **Almacenamiento:** SSD con 10+ GB de espacio libre
- **Conexión:** Internet de banda ancha (para sincronización en tiempo real)

### Requisitos de Suscripción

⚠️ **IMPORTANTE:** Algunas características requieren tipos específicos de suscripción:

- **Smart Blocks (Tech Preview):** Puede requerir activación en configuración de AutoCAD
- **Connected Support Files:** Requiere acceso a Autodesk Docs
- **Markup Import/Assist:** Requiere integración con Autodesk Docs
- **Autodesk Assistant:** Incluido en suscripciones de AutoCAD 2026
- **ArcGIS Basemaps:** Incluido en AutoCAD 2026

Verificar con el administrador de licencias de Autodesk para confirmar disponibilidad.

## Notas para Profesores

### Flexibilidad del Contenido

Este módulo está diseñado para ser **altamente editable** debido a la naturaleza evolutiva de las características de AutoCAD 2026:

1. **Actualizaciones frecuentes:** Las características de IA y cloud evolucionan rápidamente
2. **Tech Preview:** Algunas características pueden cambiar de Tech Preview a Release completo
3. **Nuevas características:** Autodesk puede agregar características adicionales en updates
4. **Cambios de interfaz:** Las UI de las características pueden modificarse

### Recomendaciones para Profesores

- ✅ **Revisar mensualmente** la documentación oficial de Autodesk
- ✅ **Probar características** antes de cada clase (pueden haber cambios)
- ✅ **Actualizar screenshots** si la interfaz cambia
- ✅ **Agregar notas** sobre cambios observados en el contenido
- ✅ **Compartir actualizaciones** con otros profesores del curso
- ✅ **Solicitar feedback** de estudiantes sobre características disponibles

### Solución de Problemas Comunes

**Problema:** Características no disponibles
- **Solución:** Verificar tipo de suscripción y activación de Tech Preview

**Problema:** Smart Blocks no funciona correctamente
- **Solución:** Verificar conexión a Internet y autenticación de Autodesk ID

**Problema:** Markup Import no sincroniza
- **Solución:** Confirmar acceso a Autodesk Docs y permisos de proyecto

**Problema:** Autodesk Assistant no responde
- **Solución:** Verificar configuración regional y disponibilidad del servicio
