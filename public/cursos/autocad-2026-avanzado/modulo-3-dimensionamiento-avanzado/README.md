# Módulo 3: Dimensionamiento Avanzado y Anotaciones

## Descripción General

Este módulo profundiza en las técnicas avanzadas de dimensionamiento y anotación en AutoCAD 2026, enfocándose en la creación de documentación técnica profesional que cumple con estándares internacionales. Aprenderás a gestionar estilos de dimensión complejos, aplicar tolerancias geométricas (GD&T), trabajar con directrices múltiples, crear tablas de datos dinámicas y utilizar campos variables para documentación automática.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- Crear y gestionar familias de estilos de dimensión avanzados
- Aplicar dimensiones especiales (angulares, radiales, ordenadas, etc.)
- Implementar tolerancias geométricas según estándares internacionales (GD&T)
- Utilizar multileaders avanzados para anotaciones profesionales
- Crear y gestionar tablas de datos con fórmulas y vínculos a Excel
- Implementar campos variables para bloques de título dinámicos
- Cumplir con estándares internacionales (ISO, ANSI, DIN, JIS, GB, GOST)

## Duración

**5-6 horas** de contenido teórico-práctico

## Estructura del Módulo

### [Lección 1: Estilos de Dimensión Avanzados](./leccion-01-estilos-dimension-avanzados.md)
**Duración: 60-75 minutos**

Domina la gestión avanzada de estilos de dimensión, incluyendo familias de estilos, dimensiones anotativas, y estándares internacionales.

**Temas principales:**
- Gestión de familias de estilos (padres e hijos)
- Sustituciones de estilos de dimensión
- Comparación de estilos
- Estándares internacionales (ISO, ANSI, DIN, JIS, GB, GOST)
- Dimensiones anotativas vs DIMSCALE
- Cortes, pliegues e inspecciones de dimensión

### [Lección 2: Dimensiones Especiales](./leccion-02-dimensiones-especiales.md)
**Duración: 50-60 minutos**

Aprende a aplicar dimensiones especiales para geometrías complejas y diseño mecánico.

**Temas principales:**
- Dimensiones angulares, radiales y de diámetro
- Dimensiones de longitud de arco
- Dimensiones con pliegue (radios grandes)
- Dimensiones ordenadas
- Dimensionamiento en cadena y paralelo
- Espaciado y alineación de dimensiones

### [Lección 3: Tolerancias de Dimensión](./leccion-03-tolerancias-dimension.md)
**Duración: 60-70 minutos**

Implementa tolerancias geométricas (GD&T) según estándares internacionales para manufactura de precisión.

**Temas principales:**
- Tolerancias geométricas (GD&T)
- Marcos de control de característica
- Símbolos de datum
- Características geométricas
- Modificadores de condición de material (MMC, LMC, RFS)
- Acumulación de tolerancias
- Estándares ISO de tolerancias

### [Lección 4: Multileaders Avanzado](./leccion-04-multileaders-avanzado.md)
**Duración: 45-55 minutos**

Domina las directrices múltiples para anotaciones profesionales en planos técnicos.

**Temas principales:**
- Opciones avanzadas de MLEADER
- Estilos de multileaders
- Estructura y formato de directrices
- Edición, alineación y recopilación de multileaders
- Multileaders anotativos
- Flujos de trabajo profesionales

### [Lección 5: Tablas de Datos](./leccion-05-tablas-datos.md)
**Duración: 55-65 minutos**

Crea tablas dinámicas con fórmulas, vínculos a Excel y extracción de datos para BOM y schedules.

**Temas principales:**
- Comando TABLE y estilos de tabla
- Formateo de datos y propiedades de celdas
- Fórmulas en tablas (SUM, AVERAGE, COUNT)
- Integración con Excel (TABLEEXPORT, DATALINK)
- Extracción de datos
- Creación de Bill of Materials (BOM)

### [Lección 6: Campos y Variables](./leccion-06-campos-variables.md)
**Duración: 40-50 minutos**

Implementa campos variables para bloques de título dinámicos y documentación automática.

**Temas principales:**
- Comando FIELD y tipos de campos
- Campos de objeto (área, longitud, etc.)
- Campos de fórmula
- Actualización de campos (automática vs manual)
- Campos en Sheet Set Manager
- Bloques de título dinámicos

### [Ejercicios Prácticos](./ejercicios.md)
**Duración: 90-120 minutos**

12-15 ejercicios avanzados que integran todos los conceptos del módulo en proyectos reales de ingeniería y arquitectura.
