# AutoCAD 2026 Avanzado

**Duración Total:** 30 horas

---

## Descripción del Curso

El curso AutoCAD 2026 Avanzado está diseñado para usuarios que han completado el nivel básico y desean profundizar en técnicas avanzadas de CAD. Este curso cubre comandos de edición complejos, gestión de bloques, dimensionamiento profesional, layouts de impresión y las características más innovadoras de AutoCAD 2026 impulsadas por inteligencia artificial.

---

## Público Objetivo

- Usuarios con conocimientos básicos de AutoCAD
- Graduados del curso AutoCAD 2026 Básico
- Profesionales que desean optimizar su flujo de trabajo CAD
- Técnicos que necesitan habilidades avanzadas de documentación
- Diseñadores que trabajan con proyectos complejos

---

## Prerrequisitos

**OBLIGATORIOS:**
- Haber completado AutoCAD 2026 Básico o equivalente
- Dominio de comandos básicos de dibujo (Line, Circle, Arc, Polyline)
- Conocimiento de capas y Object Snap
- Experiencia con comandos básicos de edición (Move, Copy, Rotate, Scale)
- Familiaridad con la interfaz de AutoCAD 2026

**RECOMENDADOS:**
- Experiencia práctica con proyectos de CAD
- Comprensión de estándares de dibujo técnico
- Conocimientos básicos de diseño arquitectónico o mecánico

## Estructura del Curso

### **Módulo 1: Edición Avanzada** (6-7 horas)

#### Temas:
- **1.1 Comando Array**
  - Rectangular Array
    - Rows y Columns
    - Spacing entre objetos
    - Angle y Base point
    - Levels (3D arrays)
  - Polar Array
    - Center point
    - Number of items
    - Angle to fill
    - Rotate items
  - Path Array
    - Path selection
    - Items y spacing
    - Align items
    - Tangent direction
  - Array editing y grip modes

- **1.2 Comando Offset**
  - Offset por distancia
  - Through point method
  - Erase y Layer options
  - Multiple offsets
  - Offset de polylines

- **1.3 Comandos Fillet y Chamfer**
  - Fillet con radius
  - Fillet Multiple y Polyline
  - Fillet Trim/No trim modes
  - Chamfer por distances
  - Chamfer por Angle
  - Chamfer Multiple

- **1.4 Comando Stretch**
  - Crossing selection requirement
  - Base point y displacement
  - Stretch vs Move
  - Stretch con múltiples objetos
  - Stretch con polylines

- **1.5 Comandos Lengthen, Join y Break**
  - Lengthen Delta
  - Lengthen Percent
  - Lengthen Total
  - Lengthen Dynamic
  - Join para líneas y arcos
  - Break at point
  - Break con 2 points

#### Ejercicios Prácticos:
- Crear patrones complejos con Array
- Diseñar componentes mecánicos con Fillet/Chamfer
- Modificar layouts arquitectónicos con Stretch
- Optimizar geometría con Lengthen y Join

---

### **Módulo 2: Bloques y Atributos** (6-7 horas)

#### Temas:
- **2.1 Creación de Bloques**
  - Block command vs Write Block (WBLOCK)
  - Block definition
  - Base point selection
  - Object selection
  - Block units
  - Explode option
  - Scale uniformly

- **2.2 Gestión de Bloques**
  - Insert block command
  - Block Editor (BEDIT)
  - Block palette
  - Redefine blocks
  - Purge unused blocks
  - Block properties

- **2.3 Bloques Dinámicos**
  - Dynamic Block Editor
  - Parameters (Linear, Polar, XY, Rotation)
  - Actions (Move, Scale, Rotate, Array)
  - Visibility States
  - Lookup Tables
  - Flip parameters
  - Alignment parameters
  - Testing dynamic blocks

- **2.4 Atributos de Bloques**
  - Attribute Definition (ATTDEF)
  - Tag, Prompt, Default value
  - Attribute modes (Invisible, Constant, Verify, Preset)
  - Edit Attributes (ATTEDIT)
  - Attribute extraction
  - Block Attribute Manager (BATTMAN)

- **2.5 Bibliotecas de Bloques**
  - Design Center para bloques
  - Tool Palettes
  - Content Explorer
  - Organizar bibliotecas personales
  - Bloques estándar de industria

#### Ejercicios Prácticos:
- Crear bloques de símbolos arquitectónicos
- Diseñar bloques dinámicos de puertas/ventanas
- Implementar atributos en title blocks
- Construir biblioteca personal de bloques

---

### **Módulo 3: Dimensionamiento Avanzado** (5-6 horas)

#### Temas:
- **3.1 Dimension Styles Avanzado**
  - Dimension Style Manager (DIMSTYLE)
  - Lines tab (extension lines, dimension lines)
  - Symbols and Arrows tab
  - Text tab (height, placement, alignment)
  - Fit tab (scale, text placement)
  - Primary Units tab
  - Alternate Units tab
  - Tolerances tab
  - Override dimension properties

- **3.2 Tipos de Dimensiones Avanzadas**
  - Ordinate dimensions
  - Arc length dimensions
  - Jogged dimensions
  - Baseline dimensions
  - Continue dimensions
  - Quick Dimension (QDIM)
  - Dimension Break

- **3.3 Multileaders (MLEADER)**
  - Multileader Style Manager
  - Leader format
  - Leader structure
  - Content (text, block, none)
  - Add/Remove leaders
  - Align multileaders
  - Collect multileaders

- **3.4 Tables**
  - Table Style Manager
  - Insert Table command
  - From data link (Excel)
  - Table editing
  - Formulas en tables
  - Table breaking
  - Cell styles y formatting

- **3.5 Fields**
  - Field command
  - Field categories (Date, Document, Objects)
  - Update fields
  - Fields en Text y MTEXT
  - Fields en Attributes
  - Automatic field updates

#### Ejercicios Prácticos:
- Crear dimension styles según estándares (ISO, ANSI)
- Dimensionar componentes mecánicos complejos
- Crear schedules con tables y data links
- Implementar title blocks con fields

---

### **Módulo 4: Layouts y Plotting** (5-6 horas)

#### Temas:
- **4.1 Model Space vs Paper Space**
  - Conceptos fundamentales
  - Layout tabs
  - Page Setup Manager
  - Layout creation
  - Layout templates

- **4.2 Viewports**
  - Creating viewports
  - Viewport scale
  - Viewport layer overrides
  - Lock viewport
  - Maximize viewport
  - Viewport properties
  - Multiple viewports

- **4.3 Plot Styles**
  - CTB (Color-based) vs STB (Named)
  - Plot Style Table Editor
  - Assign plot styles
  - Lineweights y screening
  - Color mapping

- **4.4 Plotting y Publishing**
  - Plot dialog box
  - Plot preview
  - Plot to PDF
  - Plot to DWF/DWFx
  - Batch plotting
  - Publish command
  - Plot stamp

- **4.5 Sheet Sets**
  - Sheet Set Manager
  - Create sheet set
  - Subsets organization
  - Sheet views
  - Sheet list tables
  - Transmittal packages
  - Archive sheets

#### Ejercicios Prácticos:
- Configurar layout multi-escala
- Crear plot style tables personalizados
- Configurar sheet set de proyecto
- Publicar set completo de planos

---

### **Módulo 5: Características Avanzadas AutoCAD 2026** (7-8 horas)

#### Temas:
- **5.1 Smart Blocks: Detect and Convert (AI)**
  - Introducción a Autodesk AI en AutoCAD
  - Detección automática de objetos repetitivos
  - Conversión inteligente a bloques
  - Optimización de dibujos con AI
  - Mejores prácticas para Smart Blocks
  - Limpieza automática de drawings

- **5.2 Smart Blocks: Search and Convert (AI)**
  - Búsqueda inteligente de objetos similares
  - Conversión a bloques existentes
  - Sugerencias de bloques por AI
  - Acceso a bibliotecas de bloques
  - Conversión en lote (batch conversion)
  - Gestión de bloques sugeridos

- **5.3 Smart Blocks: Replacement (AI)**
  - Reemplazo inteligente de bloques
  - Sugerencias automáticas de Autodesk AI
  - Historial de bloques recientes
  - Selección manual de bloques sustitutos
  - Actualización masiva de bloques
  - Validación de reemplazos

- **5.4 Smart Blocks: Placement (AI)**
  - Colocación automática basada en patrones
  - Aprendizaje de ubicaciones anteriores
  - Reducción de clics y tiempo
  - Detección de patrones de inserción
  - Placement suggestions
  - Optimización de flujo de trabajo

- **5.5 Autodesk Assistant (IA Conversacional)**
  - Interfaz conversacional de IA
  - Búsqueda de soluciones en tiempo real
  - Respuestas contextuales
  - Guías generadas automáticamente
  - Recursos de aprendizaje personalizados
  - Asistencia técnica instantánea
  - Tips y trucos dinámicos

- **5.6 Connected Support Files**
  - Configuración centralizada de proyectos
  - Integración con Autodesk Docs
  - Sincronización automática de configuraciones
  - Rutas de soporte conscientes del proyecto
  - Colaboración sin configuración individual
  - Gestión de archivos de soporte compartidos

- **5.7 Markup Import and Markup Assist**
  - Importación de markups desde Autodesk Docs
  - Soporte para JPG, PNG, PDF
  - Sincronización bidireccional
  - Markup Assist para incorporar cambios
  - Integración con Autodesk Docs Issues
  - Flujo de trabajo de revisión
  - Gestión de comentarios y cambios

- **5.8 What's Changed - Activity Insights**
  - Resumen de cambios por sesión
  - Lista de objetos modificados
  - Análisis de comandos utilizados
  - Cambios en tamaño de archivo
  - Tiempo total de edición
  - Cambios por usuario
  - Tracking de modificaciones
  - Informes de actividad

- **5.9 ArcGIS Basemaps**
  - Integración de mapas geoespaciales
  - Imágenes satelitales de alta resolución
  - Imágenes aéreas
  - OpenStreetMaps
  - Streets maps
  - Estilos monocromáticos (claro/oscuro)
  - Geolocalización de proyectos
  - Coordenadas geográficas

- **5.10 Hatch Improvements**
  - Patrones sin límites cerrados
  - Rellenos más flexibles
  - Texturas avanzadas
  - Gradientes y patrones personalizados
  - Hatch editing mejorado
  - Transparencia en hatches

#### Ejercicios Prácticos:
- Optimizar drawing con Smart Blocks AI
- Usar Autodesk Assistant para resolver problemas
- Configurar proyecto con Connected Support Files
- Importar y procesar markups de revisiones
- Integrar ArcGIS basemaps en site plan
- Experimentar con nuevas características de Hatch
