# Módulo 1: Edición Avanzada de Objetos

## Descripción del Módulo

Este módulo está diseñado para dominar las herramientas avanzadas de edición en AutoCAD, permitiéndote trabajar de manera más eficiente y precisa en proyectos complejos. Aprenderás a utilizar comandos que automatizan tareas repetitivas, modifican geometrías complejas y optimizan tu flujo de trabajo.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- Crear y gestionar matrices (arrays) rectangulares, polares y de trayectoria para automatizar la duplicación de objetos
- Aplicar desplazamientos (offset) precisos para crear geometrías paralelas
- Utilizar empalmes (fillet) y chaflanes (chamfer) para modificar esquinas y bordes
- Estirar y modificar longitudes de objetos manteniendo relaciones geométricas
- Unir y romper objetos para optimizar la estructura de tus dibujos
- Resolver problemas de diseño complejos combinando múltiples comandos de edición

## Prerequisitos

- Haber completado el curso **AutoCAD 2026 Básico**
- Conocimiento de comandos básicos de dibujo (LINE, CIRCLE, RECTANGLE, etc.)
- Familiaridad con métodos de selección de objetos
- Comprensión de sistemas de coordenadas y referencias a objetos (OSNAP)

## Duración Estimada

**6-7 horas** de estudio y práctica

## Estructura del Módulo

### Lección 1: Comando ARRAY - Matriz Rectangular
**Duración:** 45-50 minutos
- Comando ARRAY clásico
- Comando ARRAYRECT (matriz rectangular)
- Configuración de filas, columnas y espaciado
- Matrices tridimensionales (niveles)
- Matrices asociativas y su edición

### Lección 2: Comando ARRAY - Matriz Polar
**Duración:** 40-45 minutos
- Comando ARRAYPOLAR
- Definición de punto central y número de elementos
- Ángulo de relleno y rotación de elementos
- Punto base y sus aplicaciones
- Ejemplos prácticos (círculos de tornillos, engranajes)

### Lección 3: Comando ARRAY - Matriz de Trayectoria
**Duración:** 40-45 minutos
- Comando ARRAYPATH
- Selección de trayectorias (líneas, polilíneas, splines)
- Métodos de espaciado
- Alineación y dirección tangente
- Aplicaciones avanzadas

### Lección 4: Comando OFFSET
**Duración:** 35-40 minutos
- Comando OFFSET y alias (O)
- Distancia de desplazamiento
- Opción a través de punto
- Opciones de borrado y capa
- Desplazamiento múltiple
- Aplicaciones en arquitectura y diseño

### Lección 5: Comando FILLET
**Duración:** 45-50 minutos
- Comando FILLET y alias (F)
- Configuración de radio
- Opción de polilínea
- Modo recortar/no recortar
- Opción múltiple
- Aplicaciones en diseño mecánico y arquitectónico

### Lección 6: Comando CHAMFER
**Duración:** 40-45 minutos
- Comando CHAMFER y alias (CHA)
- Método de distancias
- Método de ángulo
- Opción de polilínea
- Comparación con FILLET
- Aplicaciones prácticas

### Lección 7: Comandos STRETCH y LENGTHEN
**Duración:** 50-55 minutos
- Comando STRETCH y alias (S)
- Selección por cruce (crossing)
- Punto base y desplazamiento
- Comando LENGTHEN y alias (LEN)
- Opciones: Delta, Porcentaje, Total, Dinámico
- Casos de uso combinados

### Lección 8: Comandos JOIN y BREAK
**Duración:** 40-45 minutos
- Comando JOIN y alias (J)
- Unión de líneas, polilíneas y arcos
- Opción cerrar
- Requisitos para unión
- Comando BREAK y alias (BR)
- Romper en punto y entre dos puntos
- Técnica BREAK con @

## Consejos para el Éxito

1. **Practica cada comando** inmediatamente después de estudiarlo
2. **Utiliza los alias** de teclado para aumentar tu velocidad
3. **Experimenta con diferentes opciones** de cada comando
4. **Combina comandos** para resolver problemas complejos
5. **Guarda tus ejercicios** para referencia futura
6. **Consulta la ayuda de AutoCAD** (tecla F1) cuando tengas dudas
