# Módulo 3: Workflows Profesionales GIS/CAD

## Descripción del Módulo

Este módulo te transforma en experto de integración GIS/CAD, dominando workflows profesionales de intercambio de datos entre Map 3D y las plataformas líderes del mercado: ArcGIS, MapInfo, AutoCAD Civil 3D, y sistemas empresariales. Aprenderás técnicas avanzadas de conversión, edición colaborativa, y gestión de proyectos multidisciplinarios.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- Integrar Map 3D con ArcGIS (Shapefiles, geodatabases, feature services)
- Trabajar con MapInfo TAB files y intercambiar datos
- Coordinar con AutoCAD Civil 3D en proyectos de infraestructura
- Realizar bulk copy y transformación masiva de datos
- Crear y administrar topologías para data quality
- Gestionar layers GIS y organización de proyectos complejos

## Duración

**6 horas** (6 lecciones de 1 hora cada una)

## Requisitos Previos

- Módulos 1 y 2 completados
- Conocimiento de ArcGIS (básico) - recomendado
- Experiencia con Civil 3D (básico) - recomendado para Lección 3

## Contenido del Módulo

### Lección 1: Integración con ESRI ArcGIS
**Duración:** 1 hora

Workflows de intercambio con ArcGIS: export/import Shapefiles, geodatabases file y enterprise, feature services, y coordination de proyectos híbridos Map 3D + ArcGIS.

**Temas clave:**
- Shapefile workflows (preserve attributes, coordinate systems)
- File geodatabase (.gdb) support
- ArcGIS Online/Portal feature services
- Best practices para data exchange

### Lección 2: Working con MapInfo TAB Files
**Duración:** 1 hora

Importación, exportación y edición de MapInfo TAB files. Workflows para proyectos que involucran stakeholders usando MapInfo Professional.

**Temas clave:**
- MapInfo TAB format (vs Shapefile)
- Import/Export TAB files en Map 3D
- Coordinate system mapping (MapInfo codes → EPSG)
- Attribute preservation

### Lección 3: Integración con AutoCAD Civil 3D
**Duración:** 1 hora

Workflows colaborativos entre Map 3D (GIS) y Civil 3D (design): compartir surfaces, alignments, corridors, y features. Project coordination para infraestructura.

**Temas clave:**
- Export Map 3D data to Civil 3D
- Import Civil 3D surfaces to Map 3D
- LandXML para exchange de alignments/surfaces
- Workflows de proyecto integrado

### Lección 4: Bulk Copy y Transformación de Datos
**Duración:** 1 hora

Técnicas avanzadas de Bulk Copy para transformación masiva de datasets: reproyección batch, conversión de formatos múltiples, filtering, y automation.

**Temas clave:**
- Bulk Copy workflow avanzado
- Batch processing de múltiples datasets
- SQL filtering durante copy
- Attribute mapping y transformation

### Lección 5: Create Topologies para Análisis
**Duración:** 1 hora

Crear topologías complejas para parcels, networks, y polygons. Advanced topology rules, validation workflows, y data quality assurance.

**Temas clave:**
- Polygon topologies (must cover, must not overlap)
- Network topologies (connectivity, flow)
- Advanced topology rules
- Automated error detection y reporting

### Lección 6: Administración de Layers GIS
**Duración:** 1 hora

Organización profesional de layers en proyectos GIS/CAD complejos: layer naming conventions, layer states, filters, y templates para consistency.

**Temas clave:**
- Layer naming standards (ISO, AIA, company standards)
- Layer states para different views
- Layer filters (by name, by property)
- Layer templates y standardization

## Ejercicios Prácticos

**6 ejercicios integradores** que simulan workflows profesionales reales:

1. ArcGIS Integration: Export Map 3D project to geodatabase para GIS analyst
2. MapInfo Collaboration: Exchange parcels data con surveyor usando MapInfo
3. Civil 3D Coordination: Share GIS utility data con Civil 3D designer
4. Bulk Copy Automation: Batch convert 20 Shapefiles a UTM con filtering
5. Topology Creation: Build parcel fabric topology y validate
6. Layer Management: Standardize 50-layer project según ISO 19100 standards
