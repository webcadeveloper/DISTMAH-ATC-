## Descripción

Curso profesional completo de AutoCAD Map 3D 2026 para dominar integración GIS/CAD, análisis espacial, workflows profesionales, y cartografía de calidad publicación. Diseñado para ingenieros civiles, topógrafos, planificadores urbanos, y especialistas GIS.

## Requisitos Previos

- Conocimientos de AutoCAD básico (dibujo 2D/3D)
- Comprensión básica de sistemas de coordenadas
- Computadora con AutoCAD 2026 (con Map 3D Toolset incluido)
- 16 GB RAM mínimo, 32 GB recomendado

## Estructura del Curso

### Módulo 1: Introducción a Map 3D y Datos GIS (6 horas)
**Fundamentos de Map 3D, sistemas de coordenadas, conexión FDO y manejo de datos geoespaciales**

- Lección 1: Interface de Map 3D y Diferencias con AutoCAD
- Lección 2: Sistemas de Coordenadas y Proyecciones
- Lección 3: Feature Data Objects (FDO) - Conexión a Datos GIS
- Lección 4: Importar/Exportar Datos GIS
- Lección 5: Geospatial Platform y Task Pane
- Lección 6: Coordinate Systems y Transformaciones
- **Ejercicios:** 6 ejercicios prácticos integradores

### Módulo 2: Análisis Espacial y Consultas (6 horas)
**Object Data, conexiones a bases de datos, consultas espaciales, topología y análisis de red**

- Lección 1: Object Data y Extended Entity Data
- Lección 2: Data Connect - Conexión a Bases de Datos
- Lección 3: Spatial Queries (Buffers, Overlays, Proximity)
- Lección 4: Topology y Relaciones Espaciales
- Lección 5: Análisis de Red (Network Analysis)
- Lección 6: Thematic Mapping y Estilos Avanzados
- **Ejercicios:** 6 ejercicios prácticos integradores

### Módulo 3: Workflows Profesionales GIS/CAD (6 horas)
**Integración con ArcGIS, MapInfo, Civil 3D, transformación de datos y topologías**

- Lección 1: Integración con ESRI ArcGIS
- Lección 2: Working con MapInfo TAB Files
- Lección 3: Integración con AutoCAD Civil 3D
- Lección 4: Bulk Copy y Transformación de Datos
- Lección 5: Create Topologies para Análisis
- Lección 6: Administración de Layers GIS
- **Ejercicios:** 6 ejercicios prácticos integradores

### Módulo 4: Cartografía y Documentación (6 horas)
**Map Books, leyendas, grids de coordenadas, estilos de plot y publicación georeferenciada**

- Lección 1: Map Books y Series de Mapas
- Lección 2: Legends, Scale Bars, North Arrows
- Lección 3: Coordinate Grids y Graticules
- Lección 4: Plot Styles para Cartografía
- Lección 5: Export to PDF/DWF con Georeferencing
- Lección 6: Publish to Autodesk Infrastructure Map Server
- **Ejercicios:** 6 ejercicios prácticos + Proyecto Final Integrador

## Objetivos de Aprendizaje

Al completar este curso, serás capaz de:

- Integrar datos GIS (SHP, KML, GeoJSON) en proyectos AutoCAD
- Configurar y trabajar con sistemas de coordenadas y proyecciones
- Realizar análisis espacial (buffers, overlays, proximity analysis)
- Conectar a bases de datos espaciales (Oracle Spatial, PostGIS)
- Crear mapas temáticos y cartografía profesional
- Implementar workflows GIS/CAD en proyectos de infraestructura
- Coordinar con ArcGIS, Civil 3D, y otros sistemas
- Crear deliverables cartográficos de calidad profesional
