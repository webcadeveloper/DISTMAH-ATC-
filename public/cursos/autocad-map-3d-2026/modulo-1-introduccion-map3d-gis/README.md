# Módulo 1: Introducción a Map 3D y Datos GIS

## Descripción del Módulo

Este módulo establece los fundamentos esenciales para trabajar con AutoCAD Map 3D 2026, la solución integrada de Autodesk que fusiona las capacidades de diseño CAD con herramientas avanzadas de Sistemas de Información Geográfica (GIS). Los estudiantes aprenderán a navegar la interfaz especializada de Map 3D, comprender los conceptos fundamentales de datos geoespaciales, y dominar las técnicas de importación/exportación de formatos GIS estándar de la industria.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- Identificar y utilizar las herramientas específicas del toolset Map 3D en AutoCAD
- Configurar y aplicar sistemas de coordenadas y proyecciones cartográficas
- Conectar a fuentes de datos GIS mediante tecnología FDO (Feature Data Objects)
- Importar y exportar datos en formatos SHP, KML, GeoJSON y LandXML
- Navegar y utilizar el Task Pane geoespacial
- Realizar transformaciones de coordenadas entre diferentes sistemas de referencia

## Duración

**6 horas** (6 lecciones de 1 hora cada una)

## Requisitos Previos

- Conocimientos de AutoCAD básico (interfaz, comandos de dibujo, layers)
- Comprensión básica de conceptos geográficos (latitud, longitud)
- AutoCAD 2026 instalado con Map 3D Toolset activado

## Contenido del Módulo

### Lección 1: Interface de Map 3D y Diferencias con AutoCAD
**Duración:** 1 hora

Introducción al entorno de Map 3D, ribbons especializados, Task Pane, y las diferencias fundamentales entre un proyecto CAD tradicional y un proyecto GIS/CAD integrado.

**Temas clave:**
- Ribbon Map 3D Toolset
- Task Pane y ventanas de herramientas
- Feature Class Manager
- Display Manager para datos geoespaciales

### Lección 2: Sistemas de Coordenadas y Proyecciones
**Duración:** 1 hora

Fundamentos de geodesia aplicada: sistemas de coordenadas geográficas vs proyectadas, datums, proyecciones cartográficas (WGS84, UTM, State Plane) y su aplicación en proyectos de ingeniería.

**Temas clave:**
- Coordinate Systems Library (EPSG codes)
- WGS84, UTM zones, State Plane Coordinate System
- Asignación de coordinate systems a dibujos
- Transformaciones de datum

### Lección 3: Feature Data Objects (FDO) - Conexión a Datos GIS
**Duración:** 1 hora

Tecnología FDO para acceder a datos geoespaciales de múltiples fuentes sin conversión. Conexión a archivos SHP, bases de datos espaciales y servicios web.

**Temas clave:**
- ¿Qué es FDO y por qué es importante?
- Data Connect workflow
- Providers FDO (SHP, SDF, Oracle, PostgreSQL)
- Feature classes y schemas

### Lección 4: Importar/Exportar Datos GIS
**Duración:** 1 hora

Workflows de importación y exportación de formatos GIS estándar de la industria: Shapefiles (SHP), KML/KMZ, GeoJSON, y LandXML para intercambio con otras plataformas.

**Temas clave:**
- Import/Export Shapefile (MAPIMPORT, MAPEXPORT)
- KML/KMZ para Google Earth
- GeoJSON para web mapping
- LandXML para diseño civil

### Lección 5: Geospatial Platform y Task Pane
**Duración:** 1 hora

Dominio del Task Pane como centro de control geoespacial: Display Manager, Data Sources, Analysis, y Map Explorer para gestión eficiente de proyectos GIS/CAD.

**Temas clave:**
- Display Manager (estilos y simbología)
- Data Sources pane
- Analysis tools
- Map Explorer hierarchy

### Lección 6: Coordinate Systems y Transformaciones
**Duración:** 1 hora

Aplicación práctica de transformaciones de coordenadas, reproyección de datos entre sistemas, y solución de problemas comunes de georreferenciación en proyectos multifuente.

**Temas clave:**
- Asignar coordinate system a drawing
- Reproyectar datos entre sistemas
- Bulk Copy con transformación
- Query con transformación on-the-fly

## Ejercicios Prácticos

Al final del módulo encontrarás **6 ejercicios integradores** que combinan los conocimientos adquiridos en escenarios profesionales reales:

1. Configuración de proyecto GIS/CAD para levantamiento topográfico
2. Importación de datos GIS municipales (SHP) y asignación de coordinate system
3. Conexión FDO a geodatabase de infraestructura
4. Transformación de coordenadas entre WGS84 y UTM para proyecto regional
5. Exportación de diseño CAD a KML para visualización en Google Earth
