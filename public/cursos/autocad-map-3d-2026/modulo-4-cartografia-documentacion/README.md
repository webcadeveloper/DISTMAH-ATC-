# Módulo 4: Cartografía y Documentación

## Descripción del Módulo

El módulo final del curso te transforma en cartógrafo profesional, dominando la creación de map books, leyendas automáticas, grids de coordenadas, estilos de plot, y publicación georeferenciada. Aprenderás a crear deliverables cartográficos de calidad profesional para presentaciones, reportes técnicos, y cumplimiento regulatorio.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- Crear Map Books (series de mapas automatizadas)
- Generar leyendas automáticas desde Display Manager styles
- Agregar scale bars, north arrows, y coordinate grids
- Configurar plot styles para cartografía profesional
- Exportar a PDF/DWF con georeferencing embebido
- Publicar a Autodesk Infrastructure Map Server (overview)

## Duración

**6 horas** (6 lecciones de 1 hora cada una)

## Requisitos Previos

- Módulos 1, 2, 3 completados
- Conocimiento de layouts en AutoCAD (PAPER SPACE)
- Comprensión de cartographic design principles (recomendado)

## Contenido del Módulo

### Lección 1: Map Books y Series de Mapas
**Duración:** 1 hora

Automatizar creación de múltiples mapas desde template: map books para atlas, tile sheets, y series de planos.

**Temas clave:**
- Map Book concept y use cases
- Define tile grid (index polygons)
- Template setup (title block, scale, etc.)
- Generate multiple sheets automáticamente

### Lección 2: Legends, Scale Bars, North Arrows
**Duración:** 1 hora

Elementos cartográficos estándar: crear leyendas automáticas desde styles, scale bars dinámicos, north arrows, y disclaimers.

**Temas clave:**
- Auto-generate legend desde Display Manager
- Dynamic scale bar (updates con viewport scale)
- North arrow placement y styles
- Cartographic text (disclaimers, sources, dates)

### Lección 3: Coordinate Grids y Graticules
**Duración:** 1 hora

Sistemas de grid de coordenadas: UTM grid, State Plane grid, lat/lon graticules, y custom grids.

**Temas clave:**
- UTM grid con labeling
- State Plane coordinate grid
- Lat/Lon graticule (grados/minutos/segundos)
- Grid spacing y formatting

### Lección 4: Plot Styles para Cartografía
**Duración:** 1 hora

Configurar CTB/STB plot styles para output profesional: pen weights, screening, color mapping para print vs digital.

**Temas clave:**
- CTB (color-based) vs STB (named) plot styles
- Lineweight mapping para clarity
- Screening (transparencies) para backgrounds
- Print vs Digital output settings

### Lección 5: Export to PDF/DWF con Georeferencing
**Duración:** 1 hora

Exportar mapas con georreferenciación embebida: PDF con coordinate info, DWF para Autodesk viewers, y best practices.

**Temas clave:**
- Georeferenced PDF (embed coordinate system)
- DWF export con metadata
- Multi-sheet PDF generation
- File size optimization

### Lección 6: Publish to Autodesk Infrastructure Map Server
**Duración:** 1 hora

Overview de publicación a Map Server (si disponible): web maps, feature services, y collaboration online.

**Temas clave:**
- Infrastructure Map Server overview
- Publish drawing as web map
- Configure feature services
- Access control y collaboration
