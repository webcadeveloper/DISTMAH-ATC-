# Módulo 1: Fundamentos de Raster Design

## Descripción del Módulo

Este módulo establece los fundamentos esenciales para trabajar con AutoCAD Raster Design 2026, el toolset especializado de Autodesk para gestión, edición y conversión de imágenes raster. Los estudiantes aprenderán a insertar y correlacionar imágenes escaneadas, fotografías aéreas y ortofotografías, aplicar técnicas de mejora de calidad, y realizar ediciones básicas que son la base para workflows profesionales de digitalización de planos legacy, facilities management y proyectos GIS.

## Objetivos de Aprendizaje

Al completar este módulo, serás capaz de:

- Identificar tipos de imágenes raster y sus características técnicas (resolución, profundidad de color, formato)
- Insertar y gestionar múltiples imágenes raster en proyectos AutoCAD
- Correlacionar imágenes escaneadas con coordenadas CAD (image correlation)
- Georreferenciar imágenes con coordenadas del mundo real
- Aplicar técnicas de mejora de calidad (brightness, contrast, despeckle, bitonal conversion)
- Realizar ediciones básicas de raster (erase, touch-up, mirror, rotate)
- Aplicar rubber sheeting para corrección de distorsiones

## Duración

**6 horas** (6 lecciones de 1 hora cada una)

## Requisitos Previos

- Conocimientos de AutoCAD básico (inserción de referencias, layers, coordenadas)
- Comprensión básica de formatos de imagen digital
- AutoCAD 2026 instalado con Raster Design Toolset activado

## Contenido del Módulo

### Lección 1: Introducción a Raster Design y Tipos de Imágenes
**Duración:** 1 hora

Fundamentos de imágenes raster, diferencias entre raster y vector, tipos de archivos (TIFF, JPG, PNG, ECW, MrSID), resolución, profundidad de color, y casos de uso profesionales de Raster Design.

**Temas clave:**
- Conceptos raster vs vector
- Formatos de imagen soportados
- Resolución y DPI
- Bitonal, grayscale, RGB, RGBA
- Activación del toolset Raster Design

### Lección 2: Inserción y Gestión de Imágenes Raster
**Duración:** 1 hora

Técnicas de inserción de imágenes, Image Manager para gestión de múltiples archivos, paths absolutos vs relativos, optimización de performance, y organización de proyectos con grandes volúmenes de imagery.

**Temas clave:**
- Comando IMAGE vs IMAGEATTACH
- Image Manager interface
- Image Frame y clipping boundaries
- Draw order y transparency
- Saved paths y path management

### Lección 3: Image Correlation y Georeferencing
**Duración:** 1 hora

Correlación de imágenes escaneadas con coordenadas CAD mediante puntos de control, georeferenciación con coordenadas del mundo real, y workflows para integración con datos GIS.

**Temas clave:**
- Correlation points (2-point, 3-point, 4-point)
- Correlation methods (displacement, rubber sheet)
- World files (.tfw, .jgw, .pgw)
- Integration con Map 3D coordinate systems
- ESRI World Files

### Lección 4: Image Enhancement - Mejora de Calidad
**Duración:** 1 hora

Técnicas de mejora de calidad de imágenes escaneadas: ajuste de brillo y contraste, eliminación de ruido (despeckle), conversión bitonal, y filtros para optimizar imágenes para vectorización.

**Temas clave:**
- Brightness y Contrast adjustment
- Despeckle (noise reduction)
- Bitonal conversion (threshold)
- Invert colors
- Histogram analysis

### Lección 5: Image Editing Básico
**Duración:** 1 hora

Herramientas de edición raster: borrado de elementos no deseados, touch-up para correcciones menores, espejado, rotación, y técnicas de limpieza de planos escaneados antes de vectorización.

**Temas clave:**
- Erase raster content
- Touchup tool
- Biased touchup (fill vs erase)
- Mirror y Rotate images
- Raster snaps

### Lección 6: Rubber Sheeting y Image Transformation
**Duración:** 1 hora

Corrección de distorsiones en imágenes mediante rubber sheeting, transformaciones geométricas, ajuste de escala no uniforme, y técnicas para alinear múltiples imágenes con diferentes distorsiones.

**Temas clave:**
- Rubber sheeting concepts
- Displacement vectors
- Triangulation methods
- Scale y skew correction
- Mosaic alignment
