# Módulo 1: Introducción a Revit MEP Mecánico

## Descripción del Módulo

Este módulo introductorio te familiarizará con la interfaz especializada de Revit MEP para sistemas mecánicos. Aprenderás a configurar proyectos mecánicos desde cero, dominar las familias MEP básicas y ajustar configuraciones específicas para instalaciones HVAC.

**Duración:** 4 horas
**Nivel:** Intermedio
**Lecciones:** 4
**Ejercicios prácticos:** 6

---

## Objetivos de Aprendizaje

Al finalizar este módulo serás capaz de:

- Navegar eficientemente por la interfaz MEP de Revit 2026
- Configurar plantillas de proyecto para sistemas mecánicos
- Identificar y utilizar familias MEP específicas para HVAC
- Ajustar settings mecánicos para optimizar el flujo de trabajo
- Comprender las mejoras de Revit 2026 para MEP mecánico
- Establecer las bases para diseño de sistemas de climatización
