# Módulo 4: Sistemas de Ventilación

## Descripción del Módulo

Este módulo se dedica a los sistemas de ventilación mecánica y extracción en edificios modernos. Aprenderás a diseñar sistemas de extracción eficientes, gestionar aire fresco y ventilación balanceada, dimensionar ductos correctamente y seleccionar ventiladores apropriados para aplicaciones específicas. Utilizarás Revit MEP 2026 para modelar estos sistemas de forma profesional.

**Duración:** 5 horas
**Nivel:** Intermedio a Avanzado
**Lecciones:** 5
**Ejercicios prácticos:** 7

---

## Objetivos de Aprendizaje

Al finalizar este módulo serás capaz de:

- Diseñar sistemas de extracción de aire completos y balanceados
- Implementar ventilación natural y mecánica según normativas
- Dimensionar conductos de ventilación con cálculos precisos
- Seleccionar ventiladores adecuados para diferentes aplicaciones
- Optimizar flujos de aire en espacios comerciales y residenciales
- Integrar sistemas de ventilación con climatización HVAC
- Cumplir con códigos de ventilación y regulaciones de aire interior
- Documentar sistemas de ventilación profesionalmente en Revit

   - Sistemas de escape de kitchenettes y baños

   - Sistemas de ventilación balanceada (HRV/ERV)
