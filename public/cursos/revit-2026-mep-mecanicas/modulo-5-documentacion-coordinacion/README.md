# Módulo 5: Documentación y Coordinación MEP

## Descripción

Este módulo final se enfoca en la **documentación profesional**, **coordinación multidisciplinaria** y preparación para **fabricación MEP** en proyectos BIM. Los estudiantes aprenderán a crear tablas de equipos personalizadas, implementar estrategias de etiquetado eficientes, coordinar modelos MEP con arquitectura y estructura, detectar interferencias con Coordination Models mejorados (Revit 2026), y preparar modelos para fabricación directa.

## Objetivos de Aprendizaje

Al completar este módulo, los estudiantes serán capaces de:

1. Crear y personalizar **tablas de equipos MEP** (schedules) con parámetros compartidos y fórmulas calculadas
2. Implementar **estrategias de etiquetado** (tags) para equipos, ductos, tuberías y dispositivos
3. Coordinar modelos MEP con **arquitectura y estructura** mediante vínculos y Copy/Monitor
4. Utilizar **Coordination Models mejorados** (Revit 2026) para gestión de interferencias
5. Ejecutar **Clash Detection** sistemático con reglas personalizadas y reportes
6. Preparar modelos MEP para **fabricación** (MEP Fabrication fundamentals)

## Contenido del Módulo

### Lecciones

1. **Tablas de Equipos MEP** - Schedules personalizados con campos calculados
2. **Etiquetado MEP Profesional** - Tags de equipos, ductos, tuberías y estrategias de visibilidad
3. **Coordinación con Arquitectura** - Vínculos, Copy/Monitor, Coordination Review
4. **Coordination Models 2026** - Mejoras en gestión de modelos coordinados
5. **Clash Detection Avanzado** - Reglas, sets, reportes e integración con BIM 360
6. **Fabricación MEP Fundamentals** - Preparación de modelos para fabricación directa

### Ejercicios Prácticos

Este módulo incluye **9 ejercicios completos** que cubren:

- Creación de tablas de equipos con fórmulas calculadas
- Etiquetado masivo de sistemas MEP completos
- Coordinación de modelo MEP con arquitectura vinculada
- Configuración de Coordination Models con múltiples disciplinas
- Ejecución de Clash Detection con reglas personalizadas
- Preparación de ductos rectangulares para fabricación CAM

## Duración

**4 horas** de contenido teórico-práctico

## Herramientas de Revit 2026

- **Schedules/Quantities** - Tablas de equipos, ductos, tuberías
- **Tags** - Etiquetas de categoría, multi-categoría y material
- **Manage Links** - Gestión de vínculos Revit y CAD
- **Copy/Monitor** - Herramienta de coordinación con notificaciones
- **Coordination Models** (mejorado 2026) - Gestión de modelos coordinados
- **Interference Check** - Detección de interferencias (clash detection)
- **MEP Fabrication Parts** - Componentes para fabricación

## Requisitos Previos

- Completar Módulos 1-4 del curso
- Conocimiento sólido de sistemas HVAC en Revit
- Familiaridad con vínculos Revit (links)
- Comprensión de flujos de trabajo BIM colaborativos
