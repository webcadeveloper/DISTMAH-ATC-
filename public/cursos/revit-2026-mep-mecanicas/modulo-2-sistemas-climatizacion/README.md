# Módulo 2: Sistemas de Climatización (HVAC)

## Descripción del Módulo

Este módulo te proporciona conocimientos avanzados en el diseño e implementación de sistemas de climatización (HVAC) utilizando Revit MEP 2026. Aprenderás a modelar difusores, rejillas, ductos, accesorios, unidades manejadoras de aire (UMA), cajas VAV y a aplicar Enhanced System Zones para análisis de zonas térmicas mejoradas.

**Duración:** 6 horas
**Nivel:** Intermedio a Avanzado
**Lecciones:** 7
**Ejercicios prácticos:** 10

---

## Objetivos de Aprendizaje

Al finalizar este módulo serás capaz de:

- Diseñar sistemas de climatización completos en Revit MEP 2026
- Modelar difusores y rejillas con parámetros precisos
- Crear y conectar redes de ductos con accesorios
- Configurar unidades manejadoras de aire (UMA) y cajas VAV
- Aplicar Enhanced System Zones para análisis de zonas térmicas
- Dimensionar sistemas de climatización según normas HVAC
- Generar tablas de equipos y etiquetado automático
- Coordinar sistemas mecánicos con otras disciplinas

## Conceptos Clave

### Sistema de Climatización
Un sistema HVAC (Heating, Ventilation and Air Conditioning) integra:
- **Generación de aire:** UMA y equipos de acondicionamiento
- **Distribución:** Red de ductos y accesorios
- **Terminales:** Difusores, rejillas y cajas VAV
- **Control:** Sistemas de regulación y automatización

### Enhanced System Zones (Revit 2026)
Las zonas del sistema mejoradas permiten:
- Límites definidos por sketch o espacios
- Mejor integración con zonas HVAC
- Visualización mediante color schemes
- Análisis de cargas más precisos

## Notas Importantes

- **Revit 2026**: Este módulo aprovecha las mejoras específicas en System Zones
- **Normas HVAC**: Se utilizan estándares ASHRAE y normas locales
- **Caudales estándar**: Rangos típicos: 4-15 L/s por persona (oficinas)
- **Dimensiones:** Respetar velocidades máximas en ductos (4-6 m/s)
- **Hardware:** Mínimo 16GB RAM para modelos complejos con múltiples zonas
