# Guía del CMS para Instructores

## Introducción
El CMS (Sistema de Gestión de Contenidos) de DISTMAH permite a los instructores crear y gestionar cursos completos, incluyendo módulos, lecciones, exámenes y contenido multimedia.

## Editor de Lecciones
El editor de lecciones es la herramienta principal para crear contenido educativo. Cuenta con las siguientes características:

### 1. Editor de Texto Enriquecido (TipTap)
- **Formato de Texto**: Negrita, Cursiva, Títulos (H1, H2, H3).
- **Listas**: Numeradas y con viñetas.
- **Multimedia**:
  - **Imágenes**: Insertar imágenes mediante URL.
  - **Videos**: Incrustar videos de YouTube o subir a Microsoft Stream.
  - **Código**: Bloques de código con resaltado de sintaxis.
  - **Citas**: Bloques de cita para destacar texto importante.

### 2. Gestión de Archivos
- **Archivos Adjuntos**: Sube PDFs, archivos DWG/RVT, y otros recursos descargables para los estudiantes.
- **Videos**: Sube videos directamente a la plataforma (integración con Microsoft Stream).

### 3. Tipos de Lecciones
- **Video**: Lección principal basada en video.
- **Lectura**: Contenido de texto extenso.
- **Ejercicio**: Instrucciones para prácticas.
- **Quiz**: Evaluaciones rápidas.
- **Clase en Vivo**: Enlace a reunión de Microsoft Teams.

## Gestión de Cursos
1. **Crear Curso**: Define título, categoría, nivel y software.
2. **Estructura**:
   - Crea **Módulos** para organizar el contenido por temas.
   - Arrastra y suelta (Drag & Drop) para reordenar módulos y lecciones.
3. **Publicación**:
   - Guarda como borrador mientras trabajas.
   - Publica cuando esté listo para los estudiantes.

## Buenas Prácticas
- **Videos**: Usa videos cortos (5-10 min) para mantener la atención.
- **Recursos**: Siempre incluye los archivos base (DWG/RVT) para que el estudiante practique.
- **Descripciones**: Añade descripciones claras a cada módulo.
